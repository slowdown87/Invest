# 第14章 资产配置与组合管理：构建属于自己的投资体系

> **本章导读**：投资的核心问题不仅是"买什么"，更是"买多少"和"怎么搭配"。资产配置是决定投资成败的关键因素。研究表明，投资收益的90%来自资产配置，只有10%来自个股选择。本章将系统讲解资产配置的理念、方法和实操，帮助你构建适合自己的投资组合。记住：好的资产配置比追求高收益更重要。

## 学习目标

- 理解资产配置的核心原理和重要意义
- 掌握风险与收益的关系，理解波动与回撤
- 学会使用经典资产配置模型
- 根据不同生命周期的风险偏好进行配置
- 掌握股票组合和基金组合的构建方法
- 理解再平衡的重要性和操作方法
- 能够根据自身情况制定资产配置方案

## 核心概念

| 术语 | 解释 |
|------|------|
| **资产配置** | 将资金分配到不同资产类别以达到风险收益平衡 |
| **相关性** | 不同资产价格变动的关联程度 |
| **波动率** | 资产价格变动的剧烈程度 |
| **最大回撤** | 从最高点到最低点的最大跌幅 |
| **再平衡** | 定期调整资产比例恢复到目标配置 |
| **夏普比率** | 每承担一单位风险获得的超额收益 |
| **分散化** | 通过配置不同资产降低组合风险 |

## 一、资产配置原理：投资的顶层设计

### 1.1 为什么要做资产配置

**资产配置的重要性**：

```
诺贝尔经济学奖得主威廉·夏普：
"资产配置是决定投资者最终风险收益结果的关键因素"

研究结论：
├── 投资收益的90%来自资产配置
├── 只有约10%来自个股选择和市场时机
└── 专业机构更重视配置而非选股

这意味着什么？
├── 即使你选股能力一般
├── 只要配置做得好
└── 也能获得不错的投资结果
```

**资产配置解决的核心问题**：

```
┌─────────────────────────────────────────────────────────────┐
│              资产配置解决的三个核心问题                      │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  问题1：收益与风险的平衡                                    │
│  ├── 高收益往往伴随高风险                                  │
│  ├── 资产配置找到适合自己的风险水平                        │
│  └── 不是追求最高收益，而是在风险可控下追求合理收益        │
│                                                             │
│  问题2：流动性与收益的平衡                                  │
│  ├── 高收益资产可能流动性差                                │

│  └── 配置不同流动性的资产                                  │
│                                                             │
│  问题3：相关性低的资产组合                                  │
│  ├── 相关性：涨跌同步程度                                  │
│  ├── 相关性低 → 东方不亮西方亮                            │

│                                                             │

```

### 1.2 分散化的数学原理

**相关性原理**：

```
资产组合的波动率计算公式：

组合方差 = Σ wi²σi² + Σ Σ wiwjρijσiσj

其中：
├── wi = 资产i的权重
├── σi = 资产i的标准差
└── ρij = 资产i和j的相关系数

关键洞察：
├── 如果ρ = 1（完全正相关）→ 组合没有分散化效果
├── 如果ρ = -1（完全负相关）→ 组合风险可以完全消除
└── 如果ρ < 1 → 组合风险小于各资产风险的加权平均
```

**分散化的效果示意**：

```
案例：股票A和股票B

股票A：
├── 预期收益：12%
├── 波动率：25%
└── 高风险高收益

股票B：
├── 预期收益：8%
├── 波动率：15%
└── 低风险低收益

不同配置比例的组合效果：

┌─────────────────────────────────────────────────────────────┐
│  配置          组合收益   组合波动率   夏普比率             │
├─────────────────────────────────────────────────────────────┤
│  100%A        12.0%      25.0%        0.36                │
│  80%A+20%B    11.2%      21.5%        0.40               │
│  60%A+40%B    10.4%      18.3%        0.43               │
│  50%A+50%B    10.0%      16.8%        0.45               │
│  40%A+60%B     9.6%      15.8%        0.46               │
│  20%A+80%B     8.8%      13.2%        0.47               │
│  100%B         8.0%      15.0%        0.33                │
└─────────────────────────────────────────────────────────────┘

结论：
├── 50%A+50%B的配置
├── 收益（10%）介于两者之间
├── 但波动率（16.8%）远低于加权平均（20%）
├── 夏普比率（0.45）也优于任一单一资产
└── 这就是分散化的威力！
```

## 二、风险与收益：理解投资的两个维度

### 2.1 风险的基本概念

**风险的定义**：

```
风险 ≠ 亏损

风险 = 结果的不确定性

风险类型：
├── 波动风险：价格上下波动
├── 下行风险：可能亏损的幅度
├── 流动性风险：变现时的损失
├── 汇率风险：汇率波动损失
├── 信用风险：对方违约损失
└── 通胀风险：购买力下降

投资中的风险：
├── 短期风险：可能亏损
├── 长期风险：购买力下降
└── 最的风险：永久性损失本金
```

**衡量风险的指标**：

```
┌─────────────────────────────────────────────────────────────┐
│              主要风险衡量指标                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 波动率（标准差）                                        │
│  ├── 衡量价格波动程度                                       │
│  ├── 年化波动率 = 日波动率 × √252                         │
│  ├── A股股票年化波动率：20-40%                            │
│  └── 指数基金年化波动率：15-25%                           │
│                                                             │
│  2. 最大回撤                                                │
│  ├── 从最高点到最低点的跌幅                                │
│  ├── 最大回撤 -50% → 需要涨100%回本                       │
│  ├── A股个股最大回撤常见 -70%~-80%                        │
│  └── 组合投资可以降低最大回撤                              │
│                                                             │
│  3. Value at Risk (VaR)                                   │
│  ├── 在一定置信水平下的最大损失                            │
│  ├── 95%VaR = -10% → 95%概率损失<10%                    │
│  └── 机构投资者常用                                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 风险与收益的关系

**经典风险收益曲线**：

```
收益与风险的权衡：

预期收益（%）
     │
 20% │  A
     │  ╱
 15% │ ╱           B
     │╱        ╱
 12% │      ╱   C
     │    ╱     ╱
 10% │  ╱   ╱
     │╱  ╱
  8% │╱  D
     │ E
     │
     └──────────────────────→ 风险（波动率%）
        5%   10%   15%   20%   25%

各点含义：
├── A点：高收益，高风险
├── B点：中等收益，中等风险
├── C点：较高收益，风险也较高
├── D点：中等收益，风险较低
├── E点：低风险，低收益

重要结论：
├── 高收益必然伴随高风险
├── 但高风险不一定带来高收益
├── 关键看风险调整后的收益
└── 夏普比率 = (收益 - 无风险收益) / 风险
```

**夏普比率的应用**：

```
夏普比率 = (组合收益 - 无风险收益) / 组合波动率

含义：
├── 每承担一单位风险，获得多少超额收益
├── 夏普比率越高越好
└── 可以比较不同组合的效率

夏普比率参考：
├── 夏普比率 > 1：优秀
├── 夏普比率 0.5-1：良好
├── 夏普比率 < 0.5：一般
└── 夏普比率 < 0：差（不如买国债）

计算示例：
├── 组合收益：15%
├── 无风险收益（国债）：3%
├── 组合波动率：12%
└── 夏普比率 = (15%-3%) / 12% = 1.0

这个夏普比率=1.0意味着：
→ 每承担1%的波动风险
→ 获得1%的超额收益
→ 效率较高
```

## 三、资产配置模型：经典配置策略详解

### 3.1 股债配置模型

**最简单的股债配置**：

```
股债配置是资产配置的基础

核心逻辑：
├── 股票：高收益，高波动
├── 债券：低收益，低波动
├── 两者相关性较低（通常-0.2~0.3）
└── 组合效果优于单一资产

经典股债配置比例：

模型1：年龄法则
├── 股票比例 = 100 - 年龄
├── 30岁：70%股票 + 30%债券
├── 50岁：50%股票 + 50%债券
└── 年龄越大越保守

模型2：60/40组合
├── 60%股票 + 40%债券
├── 经典配置，历史表现稳健
├── 年化收益约8-10%
└── 最大回撤约-20%~-30%

模型3：核心-卫星配置
├── 核心：指数基金（60-70%）
├── 卫星：主动基金/个股（10-20%）
├── 债券：20-30%
└── 兼顾指数化投资和主动管理
```

**股债配置效果验证**：

```
2010-2023年A股数据验证：

┌─────────────────────────────────────────────────────────────┐
│  配置方案          年化收益    最大回撤    夏普比率         │
├─────────────────────────────────────────────────────────────┤
│  100%沪深300      7.2%        -46%        0.25              │
│  80%股+20%债      8.1%        -38%        0.32              │
│  60%股+40%债      8.8%        -28%        0.41              │
│  50%股+50%债      9.0%        -22%        0.48              │
│  40%股+60%债      8.5%        -18%        0.52              │
│  100%债券（中债）  4.5%         -5%        0.85              │
└─────────────────────────────────────────────────────────────┘

关键发现：
├── 60%股+40%债配置
├── 收益（8.8%）接近纯股票（7.2%）
├── 但最大回撤（-28%）远小于纯股票（-46%）
├── 收益差不多，风险降低近一半
└── 这就是资产配置的价值
```

```
├── 经济增长上升
├── 经济增长下降

└── 通胀下降

资产              配置比例    风险贡献
──────────────────────────────────────

长期国债          40%        约30%
中期国债          15%        约10%
黄金              7.5%       约5%
大宗商品          7.5%       约5%
──────────────────────────────────────
合计              100%       100%

特点：
├── 各类资产风险贡献均等
├── 极端市场环境下表现相对稳健
└── 适合追求稳健收益的投资者
```

## 七、再平衡：保持组合健康的秘诀

### 7.1 什么是再平衡

**再平衡的定义**：

```
再平衡 = 将资产配置比例恢复到目标比例

为什么需要再平衡？

市场波动 → 各资产涨跌不同 → 配置比例偏离目标

举例：

初始配置：
├── 股票：60%
└── 债券：40%

一年后：
├── 股票上涨30%：60%×1.3 = 78%
└── 债券上涨5%：40%×1.05 = 42%

当前配置：
├── 股票：78%
└── 债券：42%

偏离目标：
├── 股票多了18%（78%-60%）
└── 债券少了-2%（42%-40%）

需要再平衡：
├── 卖出部分股票
├── 买入部分债券
└── 恢复到60/40
```

### 7.2 再平衡的方法与频率

**再平衡触发条件**：

```
┌─────────────────────────────────────────────────────────────┐
│              再平衡触发条件                                 │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  方法1：定期再平衡                                          │
│  ├── 频率：每年或每半年                                    │
│  ├── 优点：简单，不受情绪干扰                             │
│  ├── 缺点：可能错过极端行情                               │
│  └── 适合：大多数投资者                                    │
│                                                             │
│  方法2：阈值再平衡                                         │
│  ├── 触发条件：偏离目标超过一定幅度                        │
│  ├── 常用阈值：5%或10%                                     │
│  ├── 优点：更灵敏                                          │
│  ├── 缺点：可能频繁交易                                   │
│  └── 适合：有时间管理的投资者                              │
│                                                             │
│  方法3：混合策略                                            │
│  ├── 定期检查 + 阈值触发                                  │
│  ├── 如：每季度检查，偏离超过5%则调整                     │
│  └── 结合两者优点                                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**再平衡的具体操作**：

```
再平衡操作示例：

当前状态：
├── 股票资产：78%（目标60%）
├── 债券资产：42%（目标40%）
├── 股票超出：18%
└── 需要卖出股票，买入债券

操作步骤：
1. 计算再平衡差额
   ├── 总资产：100万
   ├── 股票目标：60万
   ├── 债券目标：40万
   └── 差额：股票18万，债券-18万

2. 执行交易
   ├── 卖出股票：18万
   └── 买入债券：18万

3. 结果
   ├── 股票：60万（60%）
   └── 债券：40万（40%）

注意事项：
├── 考虑交易成本
├── 避免频繁操作
├── 可分批执行
└── 注意税收影响
```

### 7.3 再平衡的效果

**再平衡的收益来源**：

```
再平衡的三个好处：

1. 纪律性
   ├── 强制"高卖低买"
   ├── 克服贪婪和恐惧
   └── 避免追涨杀跌

2. 风险控制
   ├── 保持目标风险水平
   ├── 避免某类资产过度集中
   └── 组合更稳健

3. 收益增强
   ├── 定期卖出的资产可能已高估
   ├── 定期买入的资产可能已低估
   └── 一定程度上实现高卖低买

历史验证：

定期再平衡 vs 不再平衡（1999-2019年）：

┌─────────────────────────────────────────────────────────────┐
│  配置方案              年化收益    最大回撤    夏普比率     │
├─────────────────────────────────────────────────────────────┤
│  60/40不再平衡         9.2%        -35%        0.51          │
│  60/40年度再平衡       9.8%        -30%        0.58          │
└─────────────────────────────────────────────────────────────┘

结论：
├── 再平衡不仅控制风险
├── 还能提升收益
└── 关键是严格执行
```

```
   ├── A股股票长期年化收益：约10%

   ├── 股票和债券相关性约0.2
   ├── 60%股票+40%债券

   ├── 大多数人无法准确择时
   ├── 即使专业投资者择时收益也有限
   ├── 但资产配置是可控的
   └── 做好配置，不需要择时

   ├── 80%的精力放在资产配置
   ├── 20%的精力选择具体标的

   ├── 设定合理的配置比例
   ├── 定期再平衡
   └── 不被市场情绪干扰

结论：
→ 资产配置是投资的"顶层设计"
→ 做好配置，投资就成功了一半
```

**题目3：什么是再平衡，为什么提升收益**

**参考答案**：

```
再平衡的定义：
再平衡是定期将资产配置比例恢复到目标水平的操作

例如：
├── 初始：60%股票 + 40%债券
├── 上涨后：75%股票 + 35%债券
├── 再平衡：卖出7.5%股票，买入7.5%债券
└── 恢复到60/40
```

**再平衡提升收益的三个原因**：

```
原因1：强制"高卖低买"

市场上涨时：
├── 股票占比提高
├── 卖出一部分股票 = 高位减仓
└── 符合"高卖"原则

市场下跌时：
├── 股票占比降低
├── 卖出一部分债券，买入股票 = 低位加仓
└── 符合"低买"原则

长期效果：
├── 在高估时降低股票
├── 在低估时增持股票
└── 被动实现逆向投资

原因2：控制风险

没有再平衡：
├── 股票持续上涨 → 占比越来越高
├── 组合风险不断累积
├── 下跌时损失惨重

定期再平衡：
├── 保持股票比例不变
├── 组合风险水平稳定
├── 避免"追涨杀跌"陷阱

原因3：克服心理偏见

人性弱点：
├── 看到涨了 → 想追加（贪婪）
├── 看到跌了 → 想卖出（恐惧）
└── 导致高买低卖

再平衡规则：
├── 制定规则，严格执行
├── 不依赖情绪判断
├── 克服人性弱点
└── 纪律创造价值

历史验证：

2010-2023年：
├── 60/40组合不再平衡：年化8.2%
├── 60/40组合年度再平衡：年化8.9%
└── 再平衡额外收益：0.7%/年

注意：
├── 再平衡也有成本（交易费用、税收）
├── 不要过于频繁（每年一次足够）
└── 关键是严格执行纪律
```
