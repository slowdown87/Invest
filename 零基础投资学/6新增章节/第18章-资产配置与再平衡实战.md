# 第18章 资产配置与再平衡实战：投资的"免费午餐"

> **本章导读**：诺贝尔经济学奖得主哈里·马科维茨曾说："资产配置是投资的唯一免费午餐。"这句话的意思是，通过合理的资产配置，可以在不增加风险的情况下提高收益，或者在不降低收益的情况下降低风险。这是投资领域最重要的智慧之一。本章将系统学习三层资金配置体系、经典配置模型、核心-卫星策略、再平衡机制，以及不同年龄段的配置方案。

---

## 学习目标

- 理解资产配置的核心原理和"免费午餐"效应
- 掌握三层资金配置体系的应用方法
- 了解经典配置模型（永久组合、股债组合、耶鲁模式等）
- 学会核心-卫星策略的构建与管理
- 掌握再平衡的触发条件和操作方法
- 能够根据自身情况制定个性化配置方案
- 理解不同年龄段的配置策略差异

---

## 核心概念

| 术语 | 解释 |
|------|------|
| **资产配置** | 根据投资目标和风险偏好，将资金分配到不同资产类别 |
| **三层资金配置** | 按资金使用时间分为短期、中期、长期三层 |
| **再平衡** | 定期调整资产比例，恢复到目标配置 |
| **核心-卫星策略** | 核心仓位配置低费率指数基金，卫星仓位配置主动或行业基金 |
| **有效前沿** | 在给定风险下收益最高（或收益一定时风险最低）的组合曲线 |
| **永久组合** | 25%股票+25%债券+25%黄金+25%现金的配置方案 |
| **股债平衡** | 股票和债券按固定比例配置 |

## 一、三层资金配置体系

### 1.1 为什么需要分层

**资金分层的重要性**：

```
┌─────────────────────────────────────────────────────────────┐
│                    资金为什么要分层？                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  核心问题：投资资金和生活资金混在一起                          │
│                                                             │
│  ❌ 没有分层的问题：                                         │
│  ├── 需要用钱时被迫卖出投资                                   │
│  ├── 可能卖在低点                                           │
│  ├── 影响投资心态                                           │
│  └── 打断复利积累                                           │
│                                                             │
│  ✅ 分层管理的优势：                                         │
│  ├── 短期资金：灵活安全，不影响投资                          │
│  ├── 中期资金：追求收益，可以承受波动                        │
│  ├── 长期资金：追求复利，承受短期波动                         │
│  └── 各司其职，互不干扰                                     │
│                                                             │
│  关键原则：                                                 │
│  ├── 用闲钱投资                                            │
│  ├── 3-5年不用的钱才做中期投资                             │
│  └── 5年以上的钱才做长期投资                                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 三层资金配置详解

**第一层：短期灵活资金**

```
┌─────────────────────────────────────────────────────────────┐
│                  第一层资金：3-6个月生活费                       │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  资金用途：                                                 │
│  ├── 覆盖3-6个月的生活支出                                  │
│  ├── 应对突发情况                                           │
│  └── 保持流动性                                             │
│                                                             │
│  配置原则：                                                 │
│  ├── 安全性第一                                            │
│  ├── 流动性第二                                            │
│  └── 收益第三                                              │
│                                                             │
│  推荐配置：                                                 │
│  ├── 货币基金：50%（随时可取）                              │
│  ├── 银行活期：30%（即时可用）                              │
│  └── 国债逆回购：20%（稍高收益）                            │
│                                                             │
│  预期收益：约1.5-2%/年                                      │
│                                                             │
│  金额计算：                                                 │
│  ├── 月支出1万 → 配置3-6万                                │
│  ├── 月支出2万 → 配置6-12万                               │
│  └── 原则：覆盖3-6个月即可                                 │
│                                                             │
│  关键：这一层资金"只进不出"，不参与投资                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**第二层：中期配置资金**

```
┌─────────────────────────────────────────────────────────────┐
│                  第二层资金：3-5年不用的钱                       │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  资金用途：                                                 │
│  ├── 3-5年内可能使用的资金                                  │
│  ├── 如买房首付、子女教育等                                 │
│  └── 追求稳健增值                                           │
│                                                             │
│  配置原则：                                                 │
│  ├── 稳健为主                                              │
│  ├── 适度追求收益                                          │
│  └── 控制波动                                              │
│                                                             │
│  推荐配置：                                                 │
│  ├── 债券基金：40%（稳健底仓）                              │
│  ├── 偏债混合基金：30%（增强收益）                          │
│  ├── 宽基指数基金：20%（适度进攻）                          │
│  └── 货币基金：10%（流动性备用）                            │
│                                                             │
│  预期收益：约5-8%/年                                       │
│                                                             │
│  关键：                                                 │
│  ├── 这层资金可以承受适度波动                               │
│  ├── 但最好在3-5年后使用                                   │
│  └── 如果提前取出可能影响收益                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**第三层：长期底仓资金**

```
┌─────────────────────────────────────────────────────────────┐
│                  第三层资金：5年以上的钱                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  资金用途：                                                 │
│  ├── 5-10年不用的钱                                       │

│  └── 传承资金                                               │
│                                                             │
│  配置原则：                                                 │
│  ├── 追求长期复利                                          │
│  ├── 承受短期波动                                           │
│  └── 分散风险                                               │
│                                                             │
│  推荐配置（年轻时）：                                        │
│  ├── 股票型基金：70%                                       │
│  │   ├── 宽基指数：50%（核心）                              │
│  │   └── 行业/风格：20%（卫星）                              │
│  ├── 债券型基金：20%                                       │
│  └── 黄金/商品：10%                                        │
│                                                             │
│  推荐配置（年长时）：                                        │
│  ├── 股票型基金：40-50%                                    │
│  ├── 债券型基金：40%                                       │
│  └── 货币/现金：10-20%                                    │
│                                                             │
│  预期收益：约8-12%/年（长期）                               │
│                                                             │
│  关键：                                                 │
│  ├── 这是复利的主要来源                                    │
│  ├── 长期不用的钱才能享受复利                              │
│  └── 不到万不得已不要取出                                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 二、经典配置模型

### 2.1 永久组合

**永久组合的由来**：

```
┌─────────────────────────────────────────────────────────────┐
│                    永久组合介绍                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  创造者：哈利·布朗（Harry Browne）                          │
│  创立时间：1970年代                                         │
│  核心理念：在任何经济环境下都能表现稳健                       │
│                                                             │
│  配置比例：                                                 │
│  ├── 股票：25%                                             │
│  ├── 债券：25%                                             │
│  ├── 黄金：25%                                             │
│  └── 现金/短期国债：25%                                    │
│                                                             │
│  设计逻辑：                                                 │
│  ├── 股票：经济增长时表现好                                 │
│  ├── 债券：经济衰退时表现好                                 │
│  ├── 黄金：通胀/动荡时表现好                               │
│  └── 现金：什么都跌时保持安全                               │
│                                                             │
│  特点：                                                     │
│  ├── 极度分散                                              │
│  ├── 任何环境都有资产表现好                                │
│  └── 适合极度保守的投资者                                  │
│                                                             │
│  历史收益：约4-5%/年（长期）                                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────────────────┐

```

## 四、再平衡机制

### 4.1 什么是再平衡

**再平衡的定义**：

```
┌─────────────────────────────────────────────────────────────┐
│                    再平衡机制解释                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  定义：定期将资产比例恢复到目标配置                          │
│                                                             │
│  例子：目标配置60/40                                        │
│                                                             │
│  期初：                                                     │
│  ├── 股票：60%（12万）                                     │
│  └── 债券：40%（8万）                                     │
│                                                             │
│  1年后（股票涨了20%，债券涨了3%）：                         │
│  ├── 股票：72%（14.4万）                                   │
│  ├── 债券：8.24万（8.24万）                                │
│  ├── 总资产：22.64万                                       │
│  ├── 实际比例：63.6%/36.4%                                │
│  └── 偏离目标：股票多了3.6%                                │
│                                                             │
│  再平衡后：                                                 │
│  ├── 卖出股票：22.64 × 3.6% = 0.81万                     │
│  ├── 买入债券：0.81万                                      │
│  ├── 股票：13.59万（60%）                                  │
│  └── 债券：9.05万（40%）                                  │
│                                                             │
│  再平衡效果：                                               │
│  ├── 强制"高卖低买"                                       │
│  ├── 控制风险                                              │
│  └── 纪律性投资                                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 4.2 再平衡的触发方式

**两种触发方式对比**：

```
┌─────────────────────────────────────────────────────────────┐
│                  再平衡触发方式                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  方式一：时间再平衡                                         │
│  ├── 定期触发（季度/半年/年）                              │
│  ├── 操作：到时即调，不考虑涨跌                            │
│  ├── 优点：简单、不需要关注市场                             │
│  ├── 缺点：可能在错误时点操作                              │
│  └── 适合：上班族、不想操心的人                            │
│                                                             │
│  方式二：阈值再平衡                                         │
│  ├── 偏离目标超过5%时触发                                  │
│  ├── 操作：达到阈值才调                                     │
│  ├── 优点：只在必要时调整                                   │
│  ├── 缺点：需要持续监控                                   │
│  └── 适合：有一定时间精力的投资者                          │
│                                                             │
│  推荐策略：                                                 │
│  ├── 主要使用时间再平衡（每半年或每年）                     │
│  ├── 辅助使用阈值再平衡（偏离>10%时强制调整）              │
│  └── 结合：每年审视 + 阈值保护                             │
│                                                             │
│  A股特别建议：                                             │
│  ├── A股波动大，阈值可以设宽一些（5-10%）                  │
│  └── 不要太频繁调整（成本高）                              │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 4.3 再平衡频率选择

**频率对比**：

```
┌─────────────────────────────────────────────────────────────┐
│                  再平衡频率对比                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  频率          操作次数（20年）    优点         缺点         │
│  ───────────────────────────────────────────────────        │
│  每月          240次              精准控制    成本高、太频繁 │
│  每季度        80次               适中         稍有滞后       │
│  每半年        40次               成本低       滞后明显       │
│  每年          20次               简单         滞后较大       │
│                                                             │
│  研究结论：                                                 │
│  ├── 再平衡频率对最终收益影响不大                           │
│  ├── 关键是坚持再平衡                                       │
│  └── 过于频繁反而增加成本                                  │
│                                                             │
│  A股推荐：                                                 │
│  ├── 每年再平衡一次（简单有效）                            │
│  ├── 配合季度检视（不调整）                                │
│  └── 偏离超过10%时临时再平衡                               │
│                                                             │
│  交易成本考量：                                             │
│  ├── ETF交易佣金：约0.03%                                 │
│  ├── 印花税（卖）：0.1%                                   │
│  └── 如果一年调整一次，成本可忽略                           │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 问答题参考答案

**题目1：资产配置的"免费午餐"效应**

**参考答案**：

```
"免费午餐"的含义：

一、马科维茨的理论贡献
├── 诺贝尔经济学奖得主哈里·马科维茨
├── 现代投资组合理论（MPT）创始人
├── 发现：通过组合投资可以"分散风险"
└── 分散化是唯一的"免费午餐"

二、免费午餐的实现
├── 相同风险下，组合收益更高
├── 相同收益下，组合风险更低
└── 关键：资产相关性低

三、数字说明
├── 股票A：收益10%，风险20%
├── 股票B：收益8%，风险15%
├── 两者相关性：0.3
│
├── 组合（各50%）：
│   ├── 收益：9%
│   ├── 风险：约14%
│   └── 风险降低了6个百分点！
│
└── 如果纯买股票B：
    ├── 收益：8%
    ├── 风险：15%
    └── 对比：组合收益更高（9% vs 8%），风险更低（14% vs 15%）

四、关键洞察
├── 不需要更高风险就能获得更高收益
├── 这就是"免费"的含义
├── 但前提：资产相关性要低
└── 相关性越低，免费午餐越丰盛

五、实践应用
├── 股债配置：股票涨时债券可能跌，反之亦然
├── 国内外配置：分散单一市场风险
├── 不同风格：价值和成长往往互补
└── 黄金配置：通胀/危机时的避风港
```

**题目5：再平衡的原理**

**参考答案**：

```
再平衡的"高卖低买"机制：

一、机制原理
├── 涨多了 → 比例超出目标 → 卖出
├── 跌多了 → 比例低于目标 → 买入
└── 结果：被动实现"高卖低买"

二、具体例子
├── 目标：股票60%，债券40%
│
├── 情景1：股票大涨
│   ├── 股票从60万涨到72万（+20%）
│   ├── 债券还是40万
│   ├── 实际比例：64.3%/35.7%
│   ├── 偏离目标4.3%
│   ├── 再平衡：卖出股票，买入债券
│   └── 结果：卖在高位！

│
├── 情景2：股票大跌
│   ├── 股票从60万跌到48万（-20%）
│   ├── 债券还是40万
│   ├── 实际比例：54.5%/45.5%
│   ├── 偏离目标5.5%
│   ├── 再平衡：卖出债券，买入股票
│   └── 结果：买在低位！

三、心理优势
├── 不需要预测市场
├── 纪律性投资
├── 克服贪婪和恐惧
└── 避免"追涨杀跌"

四、研究结论
├── 再平衡能提升收益约0.3-0.5%/年
├── 更重要的是控制风险
└── 关键是要坚持执行

五、操作提醒
├── 不要太频繁（成本）
├── 不要太迟钝（效果差）
└── 推荐：每年一次 + 阈值保护
```

**题目3：再平衡计算**

**参考答案**：

```
再平衡计算：

一、当前情况
├── 目标：股票60%，债券40%
├── 实际：股票65%，债券35%
├── 偏离：股票多5%，债券少5%
└── 总资产：100万

二、需要调整金额
├── 股票：100万 × 5% = 5万（需要卖出）
├── 债券：100万 × 5% = 5万（需要买入）

三、再平衡后
├── 股票：72万 - 5万 = 67万（67%）
├── 债券：38万 + 5万 = 43万（43%）
└── 总资产：110万

四、调整后比例
├── 股票：67/110 = 60.9% ≈ 61%
├── 债券：43/110 = 39.1% ≈ 39%
└── 非常接近目标60/40！

五、操作步骤
1. 卖出5万元股票ETF
2. 买入5万元债券基金
3. 记录本次再平衡日期和金额
```
