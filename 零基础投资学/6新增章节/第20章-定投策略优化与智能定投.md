# 第20章 定投策略优化与智能定投：普通人的"印钞机"

> **本章导读**：定投是最适合普通人的投资方式之一。它不需要预测市场，不需要择时，只需要定期投入、固定金额，长期坚持就能获得市场平均收益。巴菲特曾说："对于普通投资者，最好的策略是定投指数基金。"本章将系统学习定投的原理、微笑曲线、智能定投策略、止盈方法，以及如何构建适合自己的定投方案。

---

## 学习目标

- 理解定投的核心理念和微笑曲线原理
- 掌握智能定投的多种策略（估值定投、金字塔加仓、均线偏离）
- 学会设置止盈策略（目标止盈、估值止盈、分批止盈）
- 了解不同市场环境下定投的表现差异
- 能够根据自身情况制定定投方案
- 理解定投的常见误区和应对方法

---

## 核心概念

| 术语 | 解释 |
|------|------|
| **定投** | 定期定额投资，无论价格高低都投入固定金额 |
| **微笑曲线** | 先跌后涨的市场曲线，定投在此过程中收益丰厚 |
| **智能定投** | 根据市场估值等因素调整定投金额的策略 |
| **估值定投** | 根据PE分位调整定投金额，低位多投、高位少投 |
| **金字塔加仓** | 越跌越买，下跌时加大投入力度 |
| **止盈** | 达到收益目标后卖出，锁定利润 |
| **定投钝化** | 定投时间越长，新增资金对成本影响越小 |

### 1.2 微笑曲线原理

**微笑曲线详解**：

```
┌─────────────────────────────────────────────────────────────┐
│                    微笑曲线原理                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  什么是微笑曲线：                                           │
│  ├── 市场先跌后涨，形成"U"形                               │
│  ├── 像微笑的嘴角                                          │
│  └── 定投在此过程中收益丰厚                                │
│                                                             │
│  ┌──────────────────────────────────────────────┐          │
│  │                                              ↑          │
│  │        开始定投（1元/份）                  │          │
│  │           买100份                          │          │
│  │             ↓                               │          │
│  │    市场下跌  ████████（0.5元/份）           │          │
│  │              买200份（同样的钱买更多）       │          │
│  │                ↓                           │          │
│  │     继续下跌  ██████（0.25元/份）           │          │
│  │               买400份（更多份额！）         │          │
│  │                 ↓                          │          │
│  │      市场反弹  ██████████（0.5元/份）       │          │
│  │                 ↓                          │          │
│  │     最终上涨  ██████████████（1元/份）      │          │
│  │                                              ↓          │
│  │   成本：平均0.4元/份 → 卖出1元 = 150%收益！│          │

│                                                             │
│  为什么能赚钱？                                             │
│  ├── 下跌时：同样金额买更多份额                            │
│  ├── 积累份额：在低位积累大量低价筹码                       │
│  └── 上涨时：所有份额都在赚钱                              │
│                                                             │
│  关键数字：                                                 │
│  ├── 总投入：1000+1000+1000+1000 = 4000元                  │
│  ├── 积累份额：100+200+400+100 = 800份                     │
│  ├── 成本：4000 ÷ 800 = 5元/份                          │
│  └── 如果回到1元/份：收益100%                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 二、智能定投策略

### 2.1 估值定投法

**估值定投原理**：

```
┌─────────────────────────────────────────────────────────────┐
│                    估值定投原理                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  核心理念：低估值多买，高估值少买                          │
│                                                             │
│  操作方法：                                                 │
│  ├── 设定基准金额（如每月1000元）                          │
│  ├── 每周/每月评估指数PE分位                                │
│  ├── 根据分位调整定投金额                                   │
│  └── 分位越低，投的越多                                    │
│                                                             │
│  ┌─────────────────────────────────────────────┐           │
│  │  PE分位           定投倍数      月投入金额  │           │
│  │  ─────────────────────────────────────────│           │
│  │  <10%（极度低估）   2.0倍        2000元   │           │
│  │  10-30%（低估）     1.5倍        1500元   │           │
│  │  30-50%（正常偏低）  1.0倍        1000元   │           │
│  │  50-70%（正常偏高）  0.75倍       750元    │           │
│  │  70-90%（高估）      0.5倍        500元    │           │
│  │  >90%（极度高估）     0.25倍       250元    │           │
│  └─────────────────────────────────────────────┘           │
│                                                             │
│  数据来源：                                                 │
│  ├── 蛋卷基金：指数估值                                       │
│  ├── 天天基金：指数估值                                       │
│  ├── 支付宝：指数红绿灯                                       │
│  └── 理杏仁：指数估值历史百分位                              │
│                                                             │
│  优势：                                                     │
│  ├── 量化操作，避免主观情绪                                 │
│  ├── 在低位积累更多筹码                                     │
│  └── 在高位减少投入，控制风险                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 金字塔加仓法

**金字塔加仓原理**：

```
┌─────────────────────────────────────────────────────────────┐
│                    金字塔加仓原理                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  核心理念：越跌越买，买成金字塔                            │
│                                                             │
│  操作方法：                                                 │
│  ├── 设定基准金额（如每月1000元）                          │
│  ├── 每下跌一定幅度，增加投入                               │
│  ├── 跌得越多，买的越多                                    │
│  └── 形成金字塔式的成本结构                                │
│                                                             │
│  ┌─────────────────────────────────────────────┐           │
│  │      市场下跌幅度      投入倍数      月投入   │           │
│  │  ──────────────────────────────────────────│           │
│  │  初始定投（无跌幅）     1倍          1000元  │           │
│  │  下跌10%               1.5倍         1500元  │           │
│  │  再跌10%（累计20%）    2倍           2000元  │           │
│  │  再跌10%（累计30%）    3倍           3000元  │           │
│  │  再跌10%（累计40%）     4倍           4000元  │           │
│  │  再跌10%（累计50%）     6倍           6000元  │           │
│  └─────────────────────────────────────────────┘           │
│                                                             │
│  注意：                                                     │
│  ├── 需要预留足够资金                                       │
│  ├── 设置最大加仓倍数（如6倍）                             │
│  ├── 超过极限不再追加                                      │
│  └── 如果反弹到前期高点，停止加仓                          │
│                                                             │
│  适用场景：                                                 │
│  ├── 熊市初期或中期                                        │
│  ├── 有足够资金储备                                        │
│  └── 能够承受持续下跌的压力                                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.3 均线偏离策略

**均线偏离原理**：

```
┌─────────────────────────────────────────────────────────────┐
│                    均线偏离策略                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  核心理念：价格低于均线多买，高于均线少买                   │
│                                                             │
│  操作方法：                                                 │
│  ├── 选择均线：180日均线（适合长期定投）                    │
│  ├── 每周/每月评估当前价格vs均线                           │
│  ├── 偏离度 = (当前价格 - 均线) / 均线 × 100%              │
│  └── 根据偏离度调整定投金额                                 │
│                                                             │
│  ┌─────────────────────────────────────────────┐           │
│  │  偏离度            投入倍数      月投入      │           │
│  │  ───────────────────────────────────────────│           │
│  │  偏离>20%（高于20%）  0.5倍       500元     │           │
│  │  偏离10-20%          0.75倍       750元     │           │
│  │  偏离0-10%           1倍          1000元    │           │
│  │  偏离-10%-0%         1.5倍        1500元    │           │
│  │  偏离-20%~-10%        2倍          2000元    │           │
│  │  偏离<-20%（低于20%） 3倍          3000元    │           │
│  └─────────────────────────────────────────────┘           │
│                                                             │
│  优势：                                                     │
│  ├── 简单直观                                              │
│  ├── 有现成指标可用                                       │
│  └── 不需要估值数据                                       │
│                                                             │
│  注意：                                                     │
│  ├── 均线有滞后性                                         │
│  └── 震荡市场可能频繁切换                                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 目标止盈法

**目标止盈法详解**：

```
┌─────────────────────────────────────────────────────────────┐
│                    目标止盈法                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  方法：设定目标收益率，达到即止盈                           │
│                                                             │
│  设置参考：                                                 │
│  ├── A股波动大：目标20-30%                               │
│  ├── 成熟市场：目标10-15%                                 │
│  └── 宽基指数：目标15-25%                                 │
│                                                             │
│  实际操作：                                                 │
│  ├── 设定目标：20%                                        │
│  ├── 每周/每月检查收益率                                   │
│  ├── 达到20% → 全部赎回                                   │
│  └── 赎回后 → 立即开始新一轮定投                           │
│                                                             │
│  优点：                                                     │
│  ├── 简单直接，容易执行                                    │
│  ├── 锁定收益，落袋为安                                    │
│  └── 不需要判断市场                                        │
│                                                             │
│  缺点：                                                     │
│  ├── 可能卖早了                                          │
│  ├── 可能错过后续涨幅                                      │
│  └── 牛市止盈后可能继续涨                                 │
│                                                             │
│  优化方案：                                                 │
│  ├── 分批止盈（详见3.4节）                                │
│  ├── 动态止盈（涨越多卖越多）                             │
│  └── 梯度止盈（分批卖出不同目标）                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.3 估值止盈法

**估值止盈法详解**：

```
┌─────────────────────────────────────────────────────────────┐
│                    估值止盈法                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  方法：根据市场估值分位决定是否止盈                         │
│                                                             │
│  操作逻辑：                                                 │
│  ├── 估值分位低 → 持有/加码                               │
│  ├── 估值分位高 → 减仓/止盈                               │
│  └── 分位越高，卖的越多                                    │
│                                                             │
│  ┌─────────────────────────────────────────────┐           │
│  │  PE分位           操作           减仓比例     │           │
│  │  ───────────────────────────────────────────│           │
│  │  <30%（低估）       持有/加码     0%        │           │
│  │  30-50%（正常）     持有          0%        │           │
│  │  50-70%（正常偏高） 减仓          25%       │           │
```

### 3.4 分批止盈法

**分批止盈详解**：

```

│                    分批止盈法                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  方法：分批卖出，不一次全部止盈                             │
│                                                             │

│                                                             │

│  ├── 再卖出30%持仓                                        │
│  └── 大部分利润到手                                        │
│                                                             │
│  保留底仓（10%）：                                          │
│  └── 保留10%继续持有，享受后续涨幅                          │
│                                                             │
│  ┌─────────────────────────────────────────────┐           │
│  │  阶段    收益      卖出比例    锁定收益     │           │
│  │  ─────────────────────────────────────────  │           │
│  │  第一批  15%       30%         4.5%        │           │
│  │  第二批  25%       30%         7.5%        │           │
│  │  第三批  40%       30%        12%         │           │
│  │  底仓    收益未计   10%         -          │           │
│  │  ─────────────────────────────────────────  │           │
│  │  合计    -          -          24%         │           │
│  └─────────────────────────────────────────────┘           │
│                                                             │
│  优点：                                                     │
│  ├── 不会完全踏空                                          │
│  ├── 分批锁定收益                                          │
│  └── 心理压力小                                            │
│                                                             │
│  缺点：                                                     │
│  ├── 操作相对复杂                                          │
│  └── 可能整体收益略低于一次止盈                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**题目1：微笑曲线原理**

**参考答案**：

```
微笑曲线详解：

一、什么是微笑曲线
├── 市场价格走势呈"U"形
├── 先下跌，后上涨
├── 看起来像微笑的嘴角
└── 在此过程中定投收益丰厚

二、数字说明

假设：每月定投1000元，共定投4个月

月份    价格    定投金额    买入份额
─────────────────────────────────────
1月     1.0元   1000元     1000份
2月     0.5元   1000元     2000份
3月     0.25元  1000元     4000份
4月     0.5元   1000元     2000份
─────────────────────────────────────
合计    -       4000元     9000份

成本 = 4000 ÷ 9000 = 0.444元/份

如果第5个月回到1元/份：
├── 卖出收益：9000份 × 1元 = 9000元
├── 投入本金：4000元
├── 收益：5000元
└── 收益率：125%

三、为什么能赚钱
├── 下跌时：同样金额买更多份额
├── 低位积累：大量低价筹码
├── 上涨时：所有份额都在赚钱
└── 最终：低成本 × 大量份额 = 高收益

四、关键洞察
├── 定投赚钱不靠预测市场
├── 靠的是"跌了买更多"的机制
├── 时间越长，积累的低价筹码越多
└── 微笑曲线越大，最终收益越高

五、启示
├── 定投亏损时不要停止
├── 亏损意味着可以用更低价格买入
├── 坚持到微笑曲线右边
└── 收益会在上涨时兑现
```

### 实践题参考答案

**题目1：定投计算**

**参考答案**：

```
场景一：月定投1500元，定投10年，年化12%

公式：FV = P × [((1+r)^n - 1) / r]
其中：
├── FV = 最终资产
├── P = 每月投入
├── r = 月利率 = 年利率/12 = 12%/12 = 1%
└── n = 总月数 = 10 × 12 = 120个月

计算：
├── 累计本金 = 1500 × 120 = 18万元
├── 月利率 = 1%
├── 因子 = ((1.01)^120 - 1) / 0.01
│   ≈ (3.3 - 1) / 0.01 = 230倍
└── 最终资产 ≈ 1500 × 230 ≈ 34.5万元

收益 = 34.5 - 18 = 16.5万元
收益率 = 16.5 / 18 = 91.7%

场景二：月定投2000元，定投10年，年化12%

累计本金 = 2000 × 120 = 24万元
最终资产 ≈ 2000 × 230 = 46万元
收益 = 46 - 24 = 22万元
收益率 = 22 / 24 = 91.7%

对比：
├── 月投1500 vs 2000
├── 收益差：22 - 16.5 = 5.5万元
└── 说明：多投多赚，比例不变
```
