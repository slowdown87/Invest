## 一、财务报表基础术语

| 术语 | 英文 | 定义 | 章节 |
|------|------|------|------|
| **资产负债表** | Balance Sheet | 反映企业在某一特定日期的财务状况，包括资产、负债和所有者权益三个部分 | 第2章 |
| **利润表** | Income Statement | 反映企业在一定会计期间的经营成果，展示收入、成本和利润的关系 | 第2章 |
| **现金流量表** | Cash Flow Statement | 反映企业在一定期间现金的流入流出，包括经营、投资、筹资三类现金流 | 第2章 |
| **资产** | Asset | 企业拥有或控制的能产生经济利益的资源，包括流动资产和非流动资产 | 第2章 |
| **负债** | Liability | 企业过去的交易或事项形成的、预期会导致经济利益流出的现时义务 | 第2章 |
| **所有者权益** | Equity | 资产扣除负债后由所有者享有的剩余权益，也称净资产 | 第2章 |
| **流动资产** | Current Asset | 预计在一年或一个营业周期内变现的资产，包括现金、应收账款、存货等 | 第2章 |
| **非流动资产** | Non-current Asset | 变现周期超过一年的资产，包括固定资产、无形资产、长期投资等 | 第2章 |
| **流动负债** | Current Liability | 预计在一年内到期需要偿还的负债 | 第2章 |
| **非流动负债** | Non-current Liability | 一年以上到期才需要偿还的负债，如长期借款 | 第2章 |
| **货币资金** | Cash and Cash Equivalents | 企业库存现金和银行存款等高流动性资产 | 第2章 |
| **应收账款** | Accounts Receivable | 企业因销售商品或提供劳务而应收但尚未收到的款项 | 第2章 |
| **存货** | Inventory | 企业库存的商品、产品、原材料等 | 第2章 |
| **固定资产** | Fixed Asset | 企业使用期限超过一年的有形资产，如厂房、设备等 | 第2章 |
| **无形资产** | Intangible Asset | 没有实物形态的资产，如专利权、商标权、土地使用权 | 第2章 |
| **短期借款** | Short-term Loan | 期限在一年以内的银行借款或其他借款 | 第2章 |
| **长期借款** | Long-term Loan | 期限在一年以上的借款 | 第2章 |
| **应付账款** | Accounts Payable | 企业因购买材料、商品或接受服务而应付给供应商的款项 | 第2章 |
| **预收账款** | Advance from Customers | 企业预先收取客户的款项，代表未来的营业收入义务 | 第2章 |
| **实收资本** | Paid-in Capital | 股东实际投入的资本金 | 第2章 |
| **资本公积** | Capital Surplus | 股本溢价、资产评估增值等形成的公积金 | 第2章 |
| **盈余公积** | Surplus Reserve | 从净利润中提取的法定或任意公积金 | 第2章 |
| **未分配利润** | Retained Earnings | 企业留待以后年度分配的净利润 | 第2章 |
| **毛利润** | Gross Profit | 营业收入减去营业成本后的余额 | 第2章 |
| **净利润** | Net Profit | 收入扣除所有成本费用和所得税后的利润 | 第2章 |
| **营业收入** | Operating Revenue | 企业销售商品、提供劳务等主营业务的收入 | 第2章 |
| **营业成本** | Operating Cost | 企业已销售产品的成本 | 第2章 |
| **营业外收入** | Non-operating Income | 非日常经营活动产生的收入，如政府补助、资产处置收益 | 第2章 |
| **营业外支出** | Non-operating Expense | 非日常经营活动产生的支出 | 第2章 |
| **毛利率** | Gross Margin | 毛利润占营业收入的比例，反映产品定价能力 | 第2章、第10章 |
| **净利率** | Net Margin | 净利润占营业收入的比例，反映最终盈利水平 | 第10章 |
| **每股收益** | EPS | 普通股股东每持有一股所能享有的企业净利润 | 第11章 |
| **商誉** | Goodwill | 企业合并时支付的超过被购买方净资产公允价值的部分 | 第2章 |

## 三、估值与分析术语

| 术语 | 英文 | 定义 | 章节 |
|------|------|------|------|
| **基本面** | Fundamentals | 公司的经营状况、财务数据、行业地位等内在价值因素 | 第10章 |
| **PE（市盈率）** | P/E Ratio | 股价除以每股收益，反映回本年限 | 第5章、第11章 |
| **PB（市净率）** | P/B Ratio | 股价除以每股净资产 | 第5章、第11章 |
| **ROE** | Return on Equity | 净资产收益率，衡量股东回报率 | 第5章、第10章 |
| **ROA** | Return on Asset | 资产收益率，衡量资产使用效率 | 第10章 |
| **净利润率** | Net Profit Margin | 净利润占营业收入的比例 | 第10章 |
| **毛利率** | Gross Margin | 毛利润占营业收入的比例 | 第10章 |
| **资产负债率** | Debt-to-Asset Ratio | 总负债占总资产的比例 | 第10章 |
| **流动比率** | Current Ratio | 流动资产与流动负债之比 | 第10章 |
| **速动比率** | Quick Ratio | 速动资产与流动负债之比 | 第10章 |
| **应收账款周转率** | Receivables Turnover | 应收账款在一年内的周转次数 | 第10章 |
| **应收账款周转天数** | Receivables Days | 收回销售款项所需的平均天数 | 第10章 |
| **存货周转率** | Inventory Turnover | 存货在一年内的周转次数 | 第10章 |
| **存货周转天数** | Inventory Days | 存货从入库到售出的平均天数 | 第10章 |
| **总资产周转率** | Asset Turnover | 营业收入与平均总资产的比值 | 第10章 |
| **经营现金流** | Operating Cash Flow | 企业主营业务产生的现金流入流出净额 | 第10章 |
| **自由现金流** | Free Cash Flow | 经营现金流减去资本支出后的现金 | 第10章 |
| **现金流折现法** | DCF | 未来现金流按一定折现率计算现值的方法 | 第11章 |
| **相对估值法** | Relative Valuation | 与同类公司比较确定估值的方法 | 第11章 |
| **PEG** | PEG Ratio | 市盈率与净利润增长率的比值 | 第11章 |
| **股息率** | Dividend Yield | 年度分红金额与股价的比值 | 第11章 |
| **分红率** | Payout Ratio | 分红占净利润的比例 | 第11章 |
| **市销率** | P/S Ratio | 股价与每股销售额的比值 | 第11章 |
| **EV/EBITDA** | Enterprise Value/EBITDA | 企业价值与息税折旧摊销前利润的比值 | 第11章 |
| **息税折旧摊销前利润** | EBITDA | 未计利息、税项、折旧及摊销前的利润 | 第11章 |
| **内在价值** | Intrinsic Value | 公司基于未来现金流折现的真实价值 | 第11章 |
| **安全边际** | Margin of Safety | 股价低于内在价值的幅度 | 第11章 |
| **格雷厄姆数字** | Graham Number | 价值投资的估值公式 | 第11章 |
| **周期调整市盈率** | CAPE | 经周期性调整的PE，考虑长期平均盈利 | 第11章 |
| **企业价值** | Enterprise Value | 公司总市值加上净负债 | 第11章 |
| **商誉** | Goodwill | 收购价超出净资产公允价值的部分 | 第10章 |
| **商誉减值** | Goodwill Impairment | 商誉价值下降需要计提的损失 | 第10章 |
| **应收账款异常** | Receivable Anomaly | 应收账款增速远超收入增速的异常信号 | 第10章 |
| **存货异常** | Inventory Anomaly | 存货增速远超收入增速的异常信号 | 第10章 |
| **收入虚构** | Revenue Fictitious | 通过虚假交易虚构收入的财务造假 | 第10章 |
| **现金流异常** | Cash Flow Anomaly | 经营现金流长期低于净利润的异常 | 第10章 |
