## 一、图表-章节关联机制

### 关联设计原则

每个图表通过以下四个维度与教程关联：

1. **位置关联**：章节正文中引用图表（[图：CH12-01]）
2. **内容关联**：图表内容与章节知识点一一对应
3. **导航关联**：图表页面显示所属章节、相关章节
4. **索引关联**：README按章节/按类别双重索引

### 元数据标准

每个图表HTML文件头部包含元数据，便于追溯：

```html
<!--
图表元数据：
- 图表编号: CH12-01
- 图表名称: K线基础图解
- 所属章节: 第12章-技术分析入门
- 相关章节: 第6章-股票交易基础、第11章-估值分析入门
- 关键词: K线、技术分析、蜡烛图
- 创建日期: 2026-05-20
-->
```

## 五、使用说明

### 如何查看图表
1. **直接查看**：点击表格中的链接，直接在浏览器打开HTML文件
2. **交互功能**：所有图表支持鼠标悬停查看数据、缩放、图例切换
3. **章节关联**：图表底部显示"相关章节"导航，可跳转至教程

### 图表特点
- ✅ 使用 Chart.js 实现，数据可视化清晰
- ✅ 支持交互操作，悬停可查看具体数值
- ✅ 响应式设计，适配不同屏幕尺寸
- ✅ 中文界面，本土化友好
- ✅ A股实战案例，数据贴近中国市场
- ✅ 图表元数据完整，可追溯关联
