## 一、货币时间价值：为什么今天的钱更值钱？

### 1.1 什么是货币时间价值？

**货币时间价值**是金融学最基础的概念。简单来说：**今天拿到100元，比未来拿到100元更有价值**。

**原因有三个**：

1. **不确定性**：未来充满风险，今天的钱是确定的
2. **消费偏好**：人们普遍更愿意现在消费
3. **投资机会**：今天的钱可以立即投资获取收益

### 1.2 货币时间价值的数学表达

**终值公式**（FV, Future Value）：

```
FV = PV × (1 + r)^n

其中：
├── FV = 终值（Future Value）
├── PV = 现值（Present Value）
├── r = 年利率
└── n = 年数
```

**现值公式**（PV, Present Value）：

```
PV = FV ÷ (1 + r)^n
```

### 📊 A股案例：茅台股票的时间价值

**案例背景**：

> 2010年，贵州茅台（600519）股价约100元。假设当时用10万元买入1000股。
>
> 持有到2024年，茅台股价约1700元（后复权价格更高），10万元变成了约170万元。

**计算验证**：

```
初始投资（2010年）：10万元
持有年限：14年（2010-2024）
最终市值（2024年）：约170万元

年化收益率计算：
FV = PV × (1 + r)^n
170 = 10 × (1 + r)^14
(1 + r)^14 = 17
1 + r = 17^(1/14) ≈ 1.224
r ≈ 22.4%

茅台14年年化收益率约22.4%，远超大多数投资品种！
```

**核心启示**：优质资产的时间价值远超想象。买得好公司股票，相当于分享公司成长的复利。

## 二、单利与复利：差距有多大？

### 2.1 什么是单利和复利？

| 计算方式 | 定义 | 公式 |
|----------|------|------|
| **单利** | 只有本金产生利息 | 本息合计 = 本金 × (1 + r × n) |
| **复利** | 本金和利息都产生利息 | 本息合计 = 本金 × (1 + r)^n |

### 2.2 单利与复利对比计算

**假设条件**：本金10万元，年利率8%，投资20年

**单利计算**：

```
第1年：10万 + 0.8万 = 10.8万
第5年：10万 + (0.8万 × 5) = 14万
第10年：10万 + (0.8万 × 10) = 18万
第20年：10万 + (0.8万 × 20) = 26万
```

**复利计算**：

```
第1年：10万 × 1.08 = 10.8万
第5年：10万 × 1.08^5 = 14.7万
第10年：10万 × 1.08^10 = 21.6万
第20年：10万 × 1.08^20 = 46.6万
```

**对比表**：

| 年份 | 单利本息合计 | 复利本息合计 | 差距 |
|------|--------------|--------------|------|
| 1年 | 10.8万 | 10.8万 | 0 |
| 5年 | 14.0万 | 14.7万 | +0.7万 |
| 10年 | 18.0万 | 21.6万 | +3.6万 |
| 20年 | 26.0万 | 46.6万 | **+20.6万** |
| 30年 | 34.0万 | 100.6万 | **+66.6万** |

## 三、通胀：隐形的财富杀手

### 3.1 什么是通胀？

**通胀**（Inflation）是指物价水平持续上涨，导致货币购买力下降。简单说：**同样的钱，能买到的东西越来越少**。

**通胀率的官方数据**（2024年参考）：

| 指标 | 近年数值 |
|------|----------|
| CPI（消费者物价指数） | 约0.2%-2% |
| 官方通胀目标 | 约3% |

**注意**：官方CPI可能低估真实通胀，因为房价、教育、医疗等涨幅往往被排除在外。

### 3.2 通胀如何侵蚀购买力？

**72法则**：用72除以通胀率，可以估算购买力减半的年数。

```
72 ÷ 通胀率 = 购买力减半年数

示例：
├── 通胀率3%：72 ÷ 3 = 24年 → 24年后100元只值50元
├── 通胀率5%：72 ÷ 5 = 14.4年 → 14年后100元只值50元
└── 通胀率10%：72 ÷ 7.2年 → 7年后100元只值50元
```

### 📊 A股案例：通胀让你的股票收益"缩水"

**案例分析**：

> 2020年初买入沪深300ETF（510300），成本约4000点。
> 2024年，沪深300约3500点，看起来亏损了。
>
> 但考虑到这4年间通胀约15%，实际情况如何？

**计算对比**：

```
名义收益：3500 ÷ 4000 - 1 = -12.5%（账面亏损）

实际收益（扣除通胀后）：
├── 名义收益：-12.5%
├── 通胀影响：约-15%
└── 实际收益 ≈ -12.5% - 15% ≈ -27.5%

但如果是茅台股票：
├── 2020年初茅台约1100元
├── 2024年茅台约1700元
├── 名义收益：(1700-1100) ÷ 1100 = 54.5%
└── 实际收益：54.5% - 15% ≈ 39.5%
```

**核心结论**：
- 投资收益要跑赢通胀才算真正赚钱
- 优质股票是抵抗通胀的有效工具
- 持有现金是最差的抗通胀方式

## 四、名义利率 vs 实际利率

### 4.1 概念解析

| 利率类型 | 定义 | 公式 |
|----------|------|------|
| **名义利率** | 表面利率，未考虑通胀 | r_nominal |
| **实际利率** | 扣除通胀后的真实收益率 | r_real ≈ 名义利率 - 通胀率 |

**精确公式**：

```
实际利率 = (1 + 名义利率) ÷ (1 + 通胀率) - 1
```

**简化公式**（通胀率较低时）：

```
实际利率 ≈ 名义利率 - 通胀率
```

### 4.2 实际利率计算示例

**假设条件**：
- 银行存款利率：2.5%
- 年通胀率：3%

**计算**：

```
精确计算：
实际利率 = (1 + 2.5%) ÷ (1 + 3%) - 1
         = 1.025 ÷ 1.03 - 1
         = -0.485%

简化计算：
实际利率 ≈ 2.5% - 3% = -0.5%
```

**结论**：当名义利率 < 通胀率，实际利率为负！把钱存银行实际上是亏钱的！

## 五、机会成本：每一个决定都有代价

### 5.1 什么是机会成本？

**机会成本**是指为了得到某种东西而放弃的其他选择中，最高价值的那个选择。

> "天下没有免费的午餐"—— Milton Friedman

**简单理解**：你选择了A，就不能选B、C、D。选择A的机会成本 = B/C/D中最好的那个。

### 5.2 机会成本在投资中的应用

**案例分析**：

> 你有10万元，可以选择：
> - A：存银行定期，年利率2.5%
> - B：买国债，年利率3%
> - C：买沪深300ETF，预计年化8%

**机会成本计算**：

```
如果选择A（银行定期）：
├── 收益：2.5%
└── 机会成本：max(3%, 8%) = 8%（放弃的C选项）

如果选择C（沪深300ETF）：
├── 收益：8%
└── 机会成本：max(2.5%, 3%) = 3%（放弃的B选项）
```

**决策建议**：选择C，因为8%的收益 > 3%的机会成本。

### 📊 A股案例：机会成本的教训

**反面案例：不买房买理财**

> 2015年，小张有100万，犹豫是买房还是买理财。
>
> - 买了银行理财，年化4%，5年收益20万
> - 同期合肥房价从8000涨到20000，翻了2.5倍
>
> **机会成本**：100万 × 2.5 = 250万的房子收益 - 20万理财 = 230万

**正面案例：延迟消费投资自己**

> 2018年，小李花了2万元学习Python编程
> - 跳槽后工资从8000涨到15000
> - 月薪增加7000元，3年增加收入25万
>
> **投资回报**：25万 ÷ 2万 = 12.5倍年化收益！

## 六、折现与估值：如何计算未来的钱值多少？

### 6.1 折现的概念

**折现**（Discounting）是把未来的钱"还原"成今天的价值。这是估值的基础方法。

**为什么需要折现？**

```
未来100元 ≠ 今天100元

因为：
├── 今天的钱可以立刻投资获取收益
├── 未来有不确定性风险
└── 所以未来100元要打个"折扣"才是今天值多少
```

**折现公式**：

```
现值 = 未来金额 ÷ (1 + 折现率)^年数

PV = FV ÷ (1 + r)^n
```

### 6.2 折现计算实例

**假设**：5年后你会收到50万元，年折现率8%。

```
PV = 50万 ÷ (1 + 8%)^5
   = 50万 ÷ 1.469
   = 34.03万

结论：今天给你34万，和5年后给你50万，价值相当。
```

## 八、年金与定投：普通人如何享受复利？

### 8.1 年金的概念

**年金**是指在一定期限内，每隔相同时间收到（或付出）的等额款项。

**常见年金形式**：
- 每月工资（工作年金）
- 每月房租收入（投资年金）
- 每月房贷支出（负债年金）
- 每月定投基金（储蓄年金）

### 8.2 年金终值计算

**公式**：

```
年金终值 = 每期金额 × [(1 + r)^n - 1] ÷ r
```

**示例**：每月定投1000元，年化8%，坚持30年。

```
每期金额 = 1000元
月利率 = 8% ÷ 12 = 0.67%
期数 = 30 × 12 = 360个月

年金终值 = 1000 × [(1 + 0.67%)^360 - 1] ÷ 0.67%
         = 1000 × (11.46 - 1) ÷ 0.67%
         = 1000 × 15.6
         = 156万元

累计投入：1000 × 360 = 36万元
投资收益：156 - 36 = 120万元！
```

## 九、投资决策中的货币时间价值

### 9.1 如何比较不同投资方案？

**决策原则**：选择净现值（NPV）最大的方案。

```
净现值 = 所有未来现金流的现值之和 - 初始投资
```

**案例**：两个投资方案

> **方案A**：现在投入10万，未来5年每年返回2.5万
> **方案B**：现在投入10万，第5年末一次性返回15万
> 折现率：8%

**方案A计算**：

```
各年现金流折现：
├── 第1年：2.5万 ÷ 1.08 = 2.31万
├── 第2年：2.5万 ÷ 1.08² = 2.14万
├── 第3年：2.5万 ÷ 1.08³ = 1.98万
├── 第4年：2.5万 ÷ 1.08⁴ = 1.84万
└── 第5年：2.5万 ÷ 1.08⁵ = 1.70万

现值合计：9.97万
NPV = 9.97万 - 10万 = -0.03万（略亏）
```

**方案B计算**：

```
第5年现金流折现：15万 ÷ 1.08⁵ = 10.22万
NPV = 10.22万 - 10万 = +0.22万（微利）
```

**决策**：选择方案B，略优于方案A。

### 📊 A股实际应用：买房 vs 租房 vs 投资

**决策分析**：

> 小明有200万，首付60万买房，月供1.2万（30年还清）。
>
> 另一种选择：200万买沪深300ETF，月租3000元（与月供差额9000元用于定投）

**简化计算（30年视角）**：

```
买房方案：
├── 总支出：60万首付 + 1.2万×12×30 = 492万
├── 30年后房子价值：假设200万
└── 净成本：492 - 200 = 292万

租房+投资方案：
├── 初始投入：200万买ETF
├── 每月追加：9000元定投ETF
├── 30年后ETF市值（年化8%）：
│   ├── 初始200万：200 × 4.66 = 932万
│   └── 每月9000定投：约820万
│   └── 合计：约1752万
└── 净收益：1752万（无房贷负担）
```

**结论**：在这个案例中，长期投资股票优于买房。（注：实际情况更复杂，需考虑自住需求、租金上涨等因素）

**题目1：为什么复利被称为"世界第八大奇迹"？**

**参考答案**：

爱因斯坦是否真的说过这句话已不可考，但复利确实具有惊人的力量。

**复利的三个特点**：

**① 指数级增长**

> 线性增长：1 → 2 → 3 → 4 → 5
> 复利增长：1 → 2 → 4 → 8 → 16

前5期差距不大，但越往后差距越惊人。

**② 时间是最好的朋友**

> 10%的年收益：
> - 10年后：变成2.6倍
> - 20年后：变成6.7倍
> - 30年后：变成17.4倍
> - 40年后：变成45.3倍

**③ 微小差异，巨大差距**

> 8% vs 10%的年收益：
> - 20年后差距：6.7倍 vs 6.7倍 → 只差一点
> - 40年后差距：21.7倍 vs 45.3倍 → 差距超过2倍！

**A股案例**：

> 巴菲特的投资生涯：
> - 60年年化收益率约20%
> - 1万变成1000万，需要约50年
> - 这就是复利的威力！

**题目2：如果通胀率是5%，你希望投资收益至少达到多少？**

**参考答案**：

**计算过程**：

```
实际收益率 = 名义收益率 - 通胀率

要保持购买力不下降：
实际收益率 ≥ 0
名义收益率 - 5% ≥ 0
名义收益率 ≥ 5%
```

**最低目标**：

| 目标 | 最低名义收益率 |
|------|----------------|
| 保持购买力 | ≥ 5% |
| 实际收益5% | ≥ 10% |
| 实际收益10% | ≥ 15% |

**实际意义**：

> 如果通胀5%，你存银行拿2.5%利息，实际上每年亏损2.5%！
>
> 10年后：100元只能买到78元的东西
>
> 20年后：100元只能买到61元的东西

**结论**：在高通胀环境下，必须追求超过通胀的投资收益！

**题目4："72法则"和精确计算有什么区别？**

**参考答案**：

**72法则的适用条件**：

| 条件 | 说明 |
|------|------|
| 收益率较低 | 6%-10%区间最准确 |
| 短期计算 | 10-20年内误差较小 |
| 粗略估算 | 不需要精确值时使用 |

**72法则的误差分析**：

| 收益率 | 72法则结果 | 精确计算 | 误差 |
|--------|------------|----------|------|
| 6% | 12年 | 11.9年 | +0.1年 |
| 8% | 9年 | 9.0年 | 0 |
| 10% | 7.2年 | 7.3年 | -0.1年 |
| 12% | 6年 | 6.1年 | -0.1年 |
| 15% | 4.8年 | 5.0年 | -0.2年 |
| 20% | 3.6年 | 3.8年 | -0.2年 |

**什么时候不能用72法则**：

1. **高收益率**（>15%）：误差较大，应使用精确公式
2. **需要精确值**：如计算退休基金、保险等
3. **专业投资分析**：需要精确的NPV、IRR计算

**建议**：

> 平时快速估算用72法则
> 重要决策用精确公式
> 复杂情况用Excel或计算器

### 实践题参考答案

**题目1：10万本金，年收益率10%，20年后是多少？**

**计算过程**：

**公式法**：

```
FV = PV × (1 + r)^n
   = 10万 × (1 + 10%)^20
   = 10万 × 6.73
   = 67.3万元
```

**72法则**：

```
翻倍年数 = 72 ÷ 10 = 7.2年
20年翻倍约：2.8次（20 ÷ 7.2）
10万 × 2^2.8 ≈ 10万 × 6.96 ≈ 69.6万元
```

**结论**：

```
公式精确值：67.3万元
72法则估算：69.6万元
误差约：+2.3万（3.4%误差）
```

**题目2：10万投资选择，机会成本分析**

**分析过程**：

**选择A（银行定期2.5%）**：

**选择B（国债3%）**：
- 收益：3%
- 机会成本：max(2.5%, 8%) = 8%
- 净收益：3% - 8% = -5%（亏本）

**选择C（沪深300ETF 8%）**：
- 收益：8%
- 机会成本：max(2.5%, 3%) = 3%
- 净收益：8% - 3% = +5%（盈利）

**决策建议**：

| 选择 | 收益 | 机会成本 | 净收益 | 推荐 |
|------|------|----------|--------|------|
| A银行 | 2.5% | 8% | -5.5% | ❌ |
| B国债 | 3% | 8% | -5% | ❌ |
| C ETF | 8% | 3% | +5% | ✅ |

**结论**：选择C，机会成本最低，净收益最高。
