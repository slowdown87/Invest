# Skills 索引

按开发流程分类的技能清单。

## 使用顺序（开发流程）

```
需求 → 设计 → 实现 → 审查 → 部署
```

## 激活方式

当需要使用某个技能时，可以说：
- "激活 planning 技能"
- "用多角色模式分析"
- "帮我审查这段代码"

或者让 AI 根据场景自动选择合适的技能。
