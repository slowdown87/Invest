## 🎯 策略选择指南

### 根据资金量选择

| 资金量 | 推荐策略 | 原因 |
|--------|---------|------|
| **<5万元** | 策略1（定投） + 策略2（可转债） | 门槛低，风险小 |
| **5-20万元** | 策略1-4（定投、可转债、网格、年线） | 操作简单，稳健 |
| **20-100万元** | 策略5-8（年线、均线、共振、价值） | 进阶策略，收益更高 |
| **>100万元** | 策略8-10（价值、逆向、仓位） | 需要专业体系 |

### 根据性格选择

| 性格类型 | 推荐策略 | 原因 |
|---------|---------|------|
| **保守型** | 策略1（定投） + 策略3（高股息） | 稳健复利 |
| **平衡型** | 策略4（网格） + 策略5（年线） | 趋势+震荡 |
| **进取型** | 策略7（月周共振） + 策略8（价值投资） | 追求超额收益 |
| **逆向型** | 策略9（逆向投资） | 逆向思维 |

### 根据时间精力选择

| 投入时间 | 推荐策略 | 原因 |
|---------|---------|------|
| **<30分钟/周** | 策略1（定投） + 策略3（高股息） | 长期持有，不用盯盘 |
| **30分钟-2小时/周** | 策略4（网格） + 策略6（均线） | 条件单自动执行 |
| **>2小时/周** | 策略7（月周共振） + 策略8（价值） | 需要研究分析 |

## 🔑 核心铁律汇总

### 止盈止损铁律

| 规则 | 标准 |
|------|------|
| 止盈 | 盈利15%保留10%利润，20%为最佳止盈点 |
| 止损 | 亏损超过本金5%坚决止损 |
| 单笔亏损 | 不超过总资金5% |

### 仓位管理铁律

| 规则 | 标准 |
|------|------|
| 单只仓位 | ≤20% |
| 同行业仓位 | ≤30% |
| 保留现金 | ≥20% |
| 极端低估 | 可提升至80%-100% |

### 交易频率铁律

| 规则 | 标准 |
|------|------|
| 月交易次数 | ≤2次 |
| 持股周期 | 至少3个月以上 |
| 年换手率 | <200% |
