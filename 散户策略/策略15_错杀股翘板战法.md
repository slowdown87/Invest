## 一、零基础入门：什么是错杀股

### 1.1 用误伤理解错杀

想象你在战场：
- 友军被误伤倒下
- 战友冲上去救援
- 这就是"翘板"

**股市错杀**就是：
- 整体市场恐慌
- 好股票被拖累跌停
- 这不是真利空，是误伤
- 翘板资金进场救援

### 1.2 什么是翘板

**翘板 = 撬开跌停板**

```
极端行情：
├── 市场恐慌
├── 大量股票跌停
├── 好股被误伤
├── 资金在跌停板买入
└── 撬开跌停板

这就是"翘板"
```

### 1.3 错杀股的特征

```
被错杀的股票：
├── 基本面没有大问题
├── 没有业绩暴雷
├── 没有退市风险
├── 被市场情绪拖累
└── 有资金愿意承接

被真杀的股票：
├── 业绩大幅亏损
├── 存在退市风险
├── 有重大利空
├── 主力在出货
└── 坚决不能碰
```

## 二、专业原理：为什么错杀股有机会

### 2.1 恐慌中的机会

**极端行情往往是最佳买点**

```
恐慌杀跌时：
├── 情绪失控
├── 泥沙俱下
├── 好股坏股一起跌
├── 非理性下跌
└── 错杀机会出现

历史规律：
├── 2015年股灾：错杀股反弹100%+
├── 2018年熊市：优质股错杀后修复
├── 2020年疫情：砸出黄金坑
└── 2022年大跌：错杀股大反弹
```

### 2.2 翘板资金的逻辑

**翘板资金的动机**：

```
动机一：超跌反弹
├── 价格跌过头
├── 反弹空间大
├── 短线博弈
└── 利润丰厚

动机二：情绪修复
├── 市场恐慌过度
├── 情绪修复时涨回来
├── 跟风资金推动
└── 翘板者获利

动机三：价值发现
├── 好公司被误伤
├── 价格低于价值
├── 长线资金买入
└── 等待价值回归
```

├── 不是系统性崩盘
├── 市场情绪开始修复
├── 不是利空导致的下跌
└── 只是情绪恐慌

## 五、思考与进化：错杀战法深化

### 5.1 常见认知误区

**误区一："所有跌停都能翘"**

有重大利空的跌停是"真杀"，不能翘。

**正确做法**：只翘基本面无问题的错杀股。

**误区二："翘板是捡钱"**

翘板风险很大，成功率只有65%左右。

**正确做法**：仓位极轻，做好止损准备。

**误区三："翘板后可以持有等涨"**

翘板是短线博弈，不是价值投资。

**正确做法**：次日冲高必须走，不恋战。

**误区四："跌得越多越要补"**

越补越亏，这是散户最常犯的错误。

**正确做法**：逻辑破坏，无条件止损。

### 5.2 策略进化

**进化一：错杀+基本面**

```
基础版：纯技术面翘板
进级版：基本面无问题 + 技术面翘板

双重确认：
├── 基本面：无利空
├── 技术面：有资金承接
└── 胜率更高
```

**进化二：错杀+板块效应**

```
基础版：只看个股
进级版：个股错杀 + 板块情绪修复

判断板块：
├── 板块是否止跌
├── 是否有资金关注
├── 板块内个股表现
└── 胜率更高
```

**进化三：分批翘板**

```
激进版：一次性买入
保守版：分批买入

分批操作：
├── 撬板前试探：30%
├── 撬板成功：50%
└── 尾盘确认：20%
```

A：不行，这只是补充策略，单只仓位≤10%，总仓位≤20%。

**Q3：翘板后可以持有多久？**

A：最多3天，次日冲高必须走，不贪心。

**Q4：如果翘板失败怎么办？**

A：止损，次日低开就卖，绝不死扛。

**Q5：什么情况绝对不能翘？**

A：业绩暴雷、退市风险、连续跌停超过3个。
