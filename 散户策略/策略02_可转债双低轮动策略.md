## 一、零基础入门：什么是可转债

### 1.1 用"债券+期权"理解可转债

**可转债，全称是可转换债券**，是上市公司发行的一种特殊债券。理解它很简单：

> 想象你借给朋友10万元，朋友给你写了一张借条，承诺：
> 1. 每年给你3%的利息
> 2. 如果朋友的公司发达了，你可以把这10万元"借条"转换成公司的股票，变成股东

这就是可转债的原型：
- **债券属性**：持有到期还本付息，保本
- **股票期权**：可以转换成公司股票，享受股价上涨收益

### 1.2 可转债的三个关键价格

| 价格 | 含义 | 实战意义 |
|------|------|---------|
| **面值** | 发行时的价格，通常100元 | 可转债的"锚" |
| **到期价值** | 面值+应计利息 | 债性保护底线 |
| **市场交易价** | 实时波动的价格 | 我们的买入成本 |

### 1.3 什么是"双低"

**双低 = 低价格 + 低溢价率**

**低价格**：可转债市场交易价格低，比如90元、100元。价格越低，下跌空间越小，债性保护越强。

**低溢价率**：可转债相对于正股的"性价比"。

```
溢价率 = (可转债价格 - 转股价值) ÷ 转股价值 × 100%

转股价值 = 正股股价 ÷ 转股价 × 100
```

溢价率越低，说明可转债跟涨能力越强；溢价率越高，说明可转债跟涨越慢。

**双低策略**：同时筛选"价格低"+"溢价率低"的标的，兼顾防守与进攻。

### 1.4 零基础理解案例

你花了100元买入一只可转债：

```
情境A：股市大涨
可转债跟涨 → 价格从100涨到130 → 卖出赚30%

情境B：股市大跌
可转债有债底保护 → 价格跌到95就跌不动了 → 基本不亏

情境C：股市横盘
可转债每年拿利息 → 小幅稳定增值
```

这就是"下有保底、上不封顶"的含义。

### 2.1 可转债的"保底机制"

上市公司发行可转债的目的是**促进转股**，而不是还钱。因此，他们有强大的动力维护可转债价格：

**机制一：下调转股价**

如果股价持续低迷，上市公司可以主动下调转股价，让转股更容易。例如：

```
初始：股价10元，转股价10元 → 转股价值100元
下调后：股价8元，转股价8元 → 转股价值100元（不变）
```

**机制二：释放利好**

在可转债到期前，公司可能释放利好拉升股价，促进转股。

**机制三：资金托底**

大资金有时会在可转债价格接近面值时进场托底。

### 2.3 轮动策略的数学原理

**轮动**就是定期更换持仓，卖出价格涨高的，换入新的双低标的。

为什么轮动能赚钱？

假设你有10000元，持有两只可转债：

```
持仓1：价格95元 → 持有不动
持仓2：价格95元 → 涨到135元 → 卖出
```

卖出后，回收的135元资金用来买入新的低价格标的。

**轮动的核心逻辑**：
- 高价债：风险收益比下降 → 卖出
- 低价债：风险收益比更高 → 买入

## 三、实操流程与方法：手把手教你双低轮动

### 3.1 第一步：开通证券账户

可转债需要证券账户才能买卖：
1. 下载券商APP（如华泰、招商、国泰等）
2. 准备身份证、银行卡
3. 线上开户，约10分钟完成
4. 开通"可转债交易权限"（新规要求）

### 3.2 第二步：筛选双低标的

**双低值公式**：

```
双低值 = 可转债价格 + 溢价率 × 100
```

双低值越低越好，低于130的标的值得关注。

**筛选标准**（推荐）：

| 条件 | 数值 | 说明 |
|------|------|------|
| 可转债价格 | <130元 | 债性保护强 |
| 溢价率 | <30% | 跟涨能力强 |
| 剩余规模 | <5亿元 | 容易被炒作 |
| 评级 | AA及以上 | 违约风险低 |
| 到期时间 | >1年 | 避免短期到期压力 |

**查询工具**：

| 工具 | 网址/APP | 特点 |
|------|----------|------|
| 集思录 | jisilu.cn | 专业可转债数据 |
| 雪球 | xueqiu.com | 可转债筛选器 |
| 宁稳网 | ningwangan.com | 数据全面 |

### 3.3 第三步：构建组合

**持仓数量**：10-15只

太少：风险集中；太多：精力分散

**建仓方式**：

```
等权配置：每只可转债投入相同金额
如：10万元资金 → 10只标的 → 每只1万元
```

### 3.4 第四步：轮动规则

**轮动频率**：每2周或每月一次

**轮动触发条件**：

| 情况 | 操作 |
|------|------|
| 可转债价格>150元 | 考虑卖出，锁定利润 |
| 溢价率>50% | 考虑卖出，转股效率太低 |
| 出现新的双低标的 | 卖出弱势标的，买入新标的 |
| 突发利空 | 果断止损 |

**轮动操作示例**：

```
检查日：每月15日

持仓检查：
├── 标A：价格138元，溢价率15% → 价格偏高，考虑卖出
├── 标B：价格98元，溢价率8% → 保留
└── 标C：价格105元，溢价率45% → 溢价率过高，卖出

卖出标A和标C后，资金买入新筛选出的双低标的
```

### 3.5 第五步：止盈止损

**止盈规则**：

```
单只可转债涨到150元 → 卖出50%
单只可转债涨到170元 → 卖出剩余50%
```

**止损规则**（罕见情况）：

```
触发条件：
├── 发行主体信用评级被下调
├── 公司出现重大经营危机
└── 可转债价格跌破到期价值且逻辑破坏

操作：果断止损，避免违约风险
```

## 四、详细案例：从10万到12万的实战

### 4.1 案例背景

2024年1月，投资者小王有10万元闲钱，不想承受股票的大幅波动，但又希望获得不错的收益。了解到可转债双低策略后，开始执行。

**小王的情况**：
- 资金量：10万元
- 投资期限：1-2年
- 风险偏好：稳健型

### 4.2 建仓阶段（2024年1月）

小王在集思录筛选出12只符合条件的可转债：

| 标的 | 价格(元) | 溢价率 | 双低值 | 评级 |
|------|---------|--------|--------|------|
| A转债 | 108 | 8% | 116 | AA+ |
| B转债 | 102 | 12% | 114 | AA |
| C转债 | 115 | 5% | 120 | AAA |
| D转债 | 98 | 18% | 116 | AA+ |
| ... | ... | ... | ... | ... |

每只买入约8000元，合计持有12只，总投入约9.6万元，剩余4000元作为备用金。

### 4.3 持有与轮动（2024年1-6月）

**第一次轮动（2月）**：

```
检查发现：
├── E转债：价格涨到145元 → 卖出50%
├── F转债：价格跌到97元，溢价率飙升至55% → 逻辑破坏，卖出
└── 新标的G：价格99元，溢价率12%，双低值111 → 买入

卖出E和F后，买入G
```

**第二次轮动（3月）**：

```
检查发现：
├── H转债：正股涨停，转债价格135元 → 卖出全部
├── 新标的I：价格101元，溢价率9% → 买入
└── A转债：价格从108涨到118元 → 保留
```

### 4.4 收益统计（截至2024年12月）

**持仓情况**：

| 操作 | 数量 | 金额(元) |
|------|------|---------|
| 初始投入 | 12只 | 96,000 |

| 卖出F（亏3%） | 1只 | -290 |
| 卖出H（赚35%） | 1只 | 4,400 |
| 其他持仓变动 | - | +2,300 |
| **当前总市值** | **11只** | **108,210** |

**收益情况**：

| 指标 | 数据 |
|------|------|
| 初始资金 | 100,000元 |
| 当前市值 | 108,210元 |

**对比沪深300同期**：

| 指标 | 双低策略 | 沪深300 |

### 4.5 案例总结

**关键操作回顾**：

1. 筛选阶段：严格执行双低标准
2. 建仓阶段：分散持有12只，降低单只风险
3. 轮动阶段：每月检查，及时替换弱势标的
4. 止盈阶段：涨到目标价果断卖出
5. 风控阶段：溢价率异常时果断止损

**核心启示**：

- 双低策略在小资金量下完全可行
- 分散持有12只是合理的数量
- 轮动是超额收益的重要来源
- 最大回撤可控，心理压力小

## 五、思考与进化：双低策略的进阶之路

### 5.1 常见认知误区

**误区一："可转债不会违约"**

虽然可转债历史上违约极少，但不代表绝对不会。2022年已有可转债违约案例。

**避坑**：选择AA及以上评级，避免信用风险。

**误区二："双低值越低越好"**

双低值只是一个筛选指标，还要结合正股基本面、转债条款等综合判断。

**避坑**：双低只是门槛，还要验证正股质地。

**误区三："买了就不用管"**

可转债需要定期轮动，僵化持有可能错过机会或遭遇风险。

**避坑**：建立定期检查机制，至少每月检查一次。

### 5.2 策略进化路径

**进化一：加入正股动量因子**

基础版：只看双低值
进级版：双低值 + 正股近期涨幅 > 5%

优点：筛选出既有双低保护，又有正股动量驱动的标的

**进化二：加入条款博弈**

观察下调转股价的公告，提前埋伏
评估强赎可能性，选择强赎概率大的标的

**进化三：可转债+正股联动**

当正股出现重大利好，可转债因高溢价无法充分跟涨时，可以考虑卖出可转债买入正股

### 5.3 特殊时期的应对

**熊市时期**：
- 可转债整体价格下跌，双低机会更多
- 可以适当提高单只仓位
- 等待正股反弹带来的收益

**牛市时期**：
- 可转债普遍上涨，双低标的减少
- 降低仓位，等待新机会
- 关注可转债的止盈

**震荡市**：
- 双低策略的黄金期
- 加快轮动频率，捕捉更多机会

## 六、常见问题与避坑指南

### 6.1 Q&A

**Q1：可转债和普通债券有什么区别？**

A：普通债券持有到期还本付息；可转债多了转换为股票的权利，可以享受股价上涨收益。

**Q2：可转债会亏损吗？**

A：理论上可转债有债性保护，下跌空间有限。但如果正股持续下跌且公司出现信用问题，可转债也可能亏损。

**Q3：双低轮动需要每天盯盘吗？**

A：不需要。建议设置每周或每两周固定检查一次，平时不需要盯盘。

**Q4：有多少资金适合做双低策略？**

A：建议至少2万元以上。资金太少无法有效分散，太多了可以加大仓位。

**Q5：可转债T+0是什么意思？**

A：当天买入的可转债可以当天卖出，没有隔夜限制。相对股票T+1更灵活。

### 6.2 避坑清单

| 坑 | 描述 | 避坑方法 |
|---|------|---------|
| 重仓单只 | 单只仓位过重 | 单只不超过20% |
| 忽视评级 | 买入低评级高风险债 | 只选AA及以上 |
| 溢价过高 | 溢价率>50%不敏感 | 及时止损轮出 |
| 不知止盈 | 涨到180、200才卖 | 设定150元止盈线 |
| 盲目抄底 | 看到价格低就买 | 结合溢价率和正股 |

### 6.3 适合人群

| 适合 | 不适合 |
|------|--------|
| 稳健型投资者 | 追求暴利的激进者 |
| 资金3-5年不用的 | 短期需要用钱的 |
| 有基础投资知识的 | 完全不懂投资的 |
| 能接受小波动的 | 一点亏损都不能承受的 |
