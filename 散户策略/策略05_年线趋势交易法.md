## 一、零基础入门：什么是年线

### 1.1 用搬家理解均线

想象一个班级的平均成绩：
- 一个学生某次考100分，班级平均80分
- 下次这个学生考70分，班级平均可能还是80分

**均线就是"班级平均股价"**——把过去N天的收盘价加起来除以N，得到每天的平均股价。

### 1.2 年线是什么

**年线 = 250日均线**

```
年线 = 最近250个交易日的收盘价之和 ÷ 250
```

年线代表**市场中长期持仓者的平均成本**。为什么是250天？因为一年大约有250个交易日。

### 1.3 年线的实战意义

**三种情况**：

| 情况 | 股价位置 | 含义 |
|------|---------|------|
| 股价 > 年线 | 年线上方 | 中长期多头市场 |
| 股价 < 年线 | 年线下方 | 中长期空头市场 |
| 均线向上 | 向上发散 | 趋势向上 |
| 均线向下 | 向下发散 | 趋势向下 |

**图示**：

```
股价
  ↑
100  ┤                    ●───●───●───●
  │                ●───●
  │
 80  ┤─────────●─────────●─────────●────
  │     ●───●
  │
 60  ┤
  │
 40  ┤ ← 年线(缓慢向上)
  │
 20  ┤
  └────────────────────────────────────→ 时间
      年初      中间      年末
```

### 2.1 机构的成本线市场参与者的成本分布

A股市场的参与者：

| 参与者 | 持股周期 | 代表均线 |
|--------|---------|---------|
| 散户 | 几天-几周 | 5日线、20日线 |
| 游资 | 几周-几个月 | 60日线 |
| 机构 | 几个月-几年 | 120日线、250日线 |
| 长期投资者 | 几年以上 | 250日线 |

**年线代表机构的平均持仓成本**。当股价在年线上方，说明中长期持仓者整体盈利，市场做多情绪高涨。

### 2.2 历史数据验证

**A股三轮牛熊的年线信号**：

| 牛熊周期 | 年线上买 | 年线下卖 | 收益率 |
|---------|---------|---------|--------|
| 2005-2008 | 1000点 | 6000点 | +500% |
| 2012-2015 | 1900点 | 5100点 | +168% |
| 2018-2021 | 2450点 | 3700点 | +51% |

**完美躲过所有股灾**：
- 2008年：从6000点跌到1664点，年线下方空仓
- 2015年：股灾连续暴跌，年线下方空仓
- 2018年：从3500点跌到2440点，年线下方空仓

### 2.3 年线策略的数学优势

**年线策略 vs 买入持有**：

```
2015年牛市顶点买入，持有至今（2024年）

买入持有：
├── 买入点位：5100点
├── 当前点位：3300点
└── 收益率：-35%

年线策略：
├── 2015年跌破年线后卖出
├── 2019年站上年线后买入
├── 2022年跌破年线后卖出
└── 收益率：+60%以上
```

### 2.4 年线策略的底层逻辑

**顺势而为，不接飞刀**：

```
顺势：
├── 趋势向上 → 上涨概率 > 50%
├── 趋势向下 → 下跌概率 > 50%
└── 结论：顺势操作，胜率更高

不接飞刀：
├── 刀还在往下掉时伸手 → 接住概率低，容易受伤
├── 刀落地后再伸手 → 风险小很多
└── 年线下方不抄底 → 不接下跌的飞刀
```

## 三、实操流程与方法：手把手年线交易

### 3.1 第一步：确定标的

**年线策略的适用范围**：

**推荐操作标的**：

| 标的 | 代码 | 特点 |
|------|------|------|
| 沪深300ETF | 510310 | 稳定性强 |
| 上证50ETF | 510050 | 大盘蓝筹 |
| 创业板ETF | 159915 | 成长性强 |

### 3.2 第二步：设置信号规则

**核心信号**：

```
买入信号：
├── 条件1：沪深300指数站上年线
├── 条件2：连续3天站稳
└── 操作：分批建仓，最多80%仓位

卖出信号：
├── 条件1：沪深300指数跌破年线
├── 条件2：连续3天无法收回
└── 操作：清仓离场，不抢反弹
```

**辅助规则**：

| 规则 | 内容 |
|------|------|
| 假突破处理 | 跌破年线后3天内收回，当作有效突破；否则止损 |
| 仓位控制 | 年线上方最多80%，下方0% |
| 买入方式 | 分3批：站上年线买30%，回踩年线买30%，确认突破买20% |

### 3.3 第三步：执行买入

**建仓流程**：

```
第1步：确认信号
└── 沪深300站上年线，连续3天站稳

第2步：买入底仓
└── 买入30%仓位

第3步：等待回踩
└── 如果回踩年线且企稳，买入30%

第4步：确认突破
└── 如果继续上涨，突破前高，买入20%

第5步：持仓不动
└── 等待卖出信号
```

### 3.4 第四步：执行卖出

**卖出流程**：

```
第1步：确认信号
└── 沪深300跌破年线，连续3天无法收回

第2步：清仓离场
└── 不管盈亏，全部卖出

第3步：耐心等待
└── 不抄底，不抢反弹

第4步：等待下一个信号
└── 重新站上年线，再买入
```

### 3.5 第五步：应对假信号

**什么是假信号？**

```
真突破：站稳3天后继续上涨
假突破：站稳3天后又跌回去

处理方法：
├── 止损线设在买入价的-5%
├── 触及止损线，无条件卖出
└── 不抱有侥幸心理
```

## 五、思考与进化：年线策略的进阶

### 5.1 常见认知误区

**误区一："年线信号太滞后"**

年线确实滞后，但滞后也意味着稳定。宁可少赚一点，也要避免大亏。

**避坑**：接受年线的滞后性，不追求精确抄底逃顶。

**误区二："跌破年线可以抄底"**

很多人在跌破年线后抄底，结果被套在半山腰。

**避坑**：年线下方不抄底，让"刀"先落地。

**误区三："来回穿越年线频繁交易"**

震荡市中，指数可能多次穿越年线，产生频繁交易。

**避坑**：加上"连续3天站稳"的条件，过滤假信号。

### 5.2 策略进化

**进化一：年线+均线多空排列**

基础版：只看年线
进级版：年线向上 + 5日线上穿年线 → 买入信号更强

**进化二：年线+成交量确认**

```
有效突破特征：
├── 指数站上年线
├── 成交量放大（比前5日均量多50%以上）
└── 连续3天站稳

无效突破特征：
├── 指数站上年线
├── 成交量萎缩
└── 很快又跌回去
```

**进化三：年线+MACD背离**

```
底背离：价格创新低，MACD没有创新低
└── 预示下跌动能减弱，可能见底

顶背离：价格创新高，MACD没有创新高
└── 预示上涨动能减弱，可能见顶
```

### 5.3 结合其他策略

**年线+定投**：

```
年线上方：减少定投金额
年线下方：加倍定投金额

优势：
├── 顺风时少买，避免买在高位
└── 逆风时多买，增加积累份额
```

**年线+网格**：

```
年线上方：暂停网格，改用趋势策略
年线下方：开始网格交易，捕捉震荡收益

优势：不同行情用不同策略
```

## 六、常见问题与避坑指南

### 6.1 Q&A

**Q1：年线策略适合个股吗？**

A：也可以用，但个股更容易被操控。可以优先考虑宽基指数ETF。

**Q2：年线参数可以调整吗？**

A：可以。120日线（半年线）更灵敏，250日线更稳定。激进型可以用120日线，稳健型用250日线。

**Q3：假信号怎么处理？**

A：设置止损线，如-5%或-8%。触及止损无条件卖出。

**Q4：年线下方可以买吗？**

A：不建议。年线下方是空头市场，下跌概率大于上涨概率。

**Q5：年线策略需要每天盯盘吗？**

A：不需要。每周检查一次即可。关键是出现信号时执行。

### 6.2 避坑清单

| 坑 | 描述 | 避坑方法 |
|---|------|---------|
| 抄底心态 | 年线下方想抄底 | 年线下方空仓等待 |
| 频繁交易 | 震荡市来回穿越 | 连续3天才确认 |
| 不止损 | 假突破死扛 | 设-8%止损线 |
| 仓位过重 | 刚站上年线就满仓 | 分批建仓 |
| 预判顶底 | 想卖在最高、买在最低 | 接受模糊，只求模糊的正确 |

### 6.3 适合人群

| 适合 | 不适合 |
|------|--------|
| 追求稳健的投资者 | 追求暴利的 |
| 没时间研究个股的 | 喜欢频繁交易的 |
| 能接受"少赚一点"的 | 必须卖在最高点的 |
| 资金量较大的 | 资金量太小的（手续费占比高) |
