## 一、零基础入门：什么是网格交易

### 1.1 用渔网理解网格

想象渔民撒网捕鱼：
- 渔网有大小均匀的网眼
- 不管鱼从哪个位置游过，都会被网住

**网格交易就像一张"金融渔网"**：
- 把价格波动区间切成若干"格子"
- 价格跌到某个格子就买
- 价格涨到某个格子就卖
- 每次买卖都能赚钱

### 1.2 网格交易的原理

**核心逻辑**：在震荡行情中，价格会在一个区间内反复波动。网格交易利用这种波动，低买高卖，积小胜为大胜。

**示意图**：

```
价格
  ↑
7元  ┤ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┤ 卖出第3格
  │                       │
6元  ┤ ─ ─ ─ ─ ─ ─ ─ ─ ┤ 卖出第2格
  │                       │
5元  ┤ ─ ─ ─ ─ ─ ─ ─ ─ ┤ 基准价(中轴)
  │                       │
4元  ┤ ─ ─ ─ ─ ─ ─ ─ ─ ┤ 买入第1格
  │                       │
3元  ┤ ─ ─ ─ ─ ─ ─ ─ ─ ┤ 买入第2格
  │                       │
2元  ┤ ─ ─ ─ ─ ─ ─ ─ ─ ┤ 买入第3格
  ↓
```

**操作规则**：
- 下跌一格 → 买入
- 上涨一格 → 卖出

### 1.3 网格 vs 其他策略

| 策略 | 特点 | 适合行情 |
|------|------|---------|
| 网格交易 | 自动化高抛低吸 | **震荡市** |
| 趋势跟踪 | 追涨杀跌 | 单边趋势 |
| 定投 | 越跌越买 | 所有行情 |
| 高股息 | 长期持有 | 所有行情 |

**网格交易的核心优势**：震荡行情中自动盈利，不需要判断方向。

### 2.1 震荡市的数学优势

**关键数据**：A股70%的时间处于震荡行情。

**震荡市的特征**：
```
涨上去 → 跌下来 → 又涨上去 → 又跌下来
```

网格交易在每次波动中都能盈利：
- 上涨时：从下往上，每格都卖
- 下跌时：从上往下，每格都买

### 2.2 单边市的处理

网格交易在单边市会失效：

**单边上涨**：
```
问题：卖光了，手里没有筹码
解决：设置网格上限，触及后停止卖出
```

**单边下跌**：
```
问题：买满了，没有资金继续加仓
解决：控制总仓位，预留资金
```

### 2.3 网格的收益率计算

**案例**：

```
网格设置：
├── 基准价：5元
├── 网格间距：5%（每格涨跌0.25元）
├── 网格数量：10格（4元-6元区间）
└── 每格交易量：1000股

盈利计算：
├── 每格买卖价差：0.25元
├── 每格收益：0.25 × 1000 = 250元
├── 10格全走一遍收益：250 × 10 = 2500元
└── 如果行情震荡10次：2500 × 10 = 25000元
```

## 三、实操流程与方法：手把手搭建网格

### 3.1 第一步：选择标的

**最适合网格的标的**：

| 条件 | 标准 | 原因 |
|------|------|------|
| 波动性 | 中等偏低 | 波动太大容易卖光/买满 |
| 趋势性 | 弱 | 趋势强不适合网格 |
| 流动性 | 高 | 方便进出 |
| 稳定性 | 高 | 不怕退市 |

**推荐标的**：

| 类型 | 示例 | 优点 |
|------|------|------|
| 宽基ETF | 510310（沪深300ETF） | 稳定，不会死 |
| 行业ETF | 512880（证券ETF） | 波动适中 |
| 蓝筹股 | 贵州茅台 | 稳定性强 |
| 高股息股 | 长江电力 | 震荡区间清晰 |

**不建议网格的标的**：
- 垃圾股（可能退市）
- 庄股（容易被操控）
- 暴涨暴跌的题材股

### 3.2 第二步：设置网格参数

**核心参数**：

| 参数 | 设置方法 | 示例 |
|------|---------|------|
| 基准价 | 当前价格 | 5元 |
| 网格间距 | 3%-10% | 5% |
| 网格数量 | 5-20格 | 10格 |
| 每格仓位 | 总仓位÷格数 | 10% |

**参数选择原则**：

```
间距太小（<3%）：
├── 优点：盈利次数多
├── 缺点：手续费占比高，容易失效

间距太大（>10%）：
├── 优点：手续费占比低
├── 缺点：盈利机会少
```

**推荐间距**：
- ETF：5%
- 蓝筹股：5%-8%
- 高波动股票：8%-10%

### 3.3 第三步：确定仓位

**仓位控制公式**：

```
最大仓位 = 50%（预留50%资金应对下跌）
每格仓位 = 最大仓位 ÷ 网格数量

示例：
├── 最大仓位：50%
├── 网格数量：10格
└── 每格仓位：5%
```

**为什么只50%仓位？**

```
极端情况：价格从基准价跌到最低
├── 如果100%仓位：亏50%，可能爆仓
└── 如果50%仓位：亏25%，还有翻身机会
```

### 3.4 第四步：执行网格交易

**交易规则**：

```
场景1：价格下跌
├── 跌到第1格价格 → 买入1格仓位
├── 继续跌到第2格价格 → 再买1格
└── 跌到最低格 → 不再买入，等待反弹

场景2：价格上涨
├── 涨到第1格价格 → 卖出1格
├── 继续涨到第2格价格 → 再卖1格
└── 涨到最高格 → 不再卖出，等待下跌

场景3：回到基准价
└── 回到原点，仓位归零，继续等待
```

### 3.5 第五步：使用券商条件单

**华宝证券条件单设置**：

```
网格条件单设置：
1. 标的：510310（沪深300ETF）
2. 基准价：5.00元
3. 触发价格：±5%（4.75-5.25元）
4. 每笔委托：1000股
5. 截止日期：长期有效
6. 回落卖出 + 反弹买入：勾选
```

**自动网格优势**：
- 无需盯盘
- 自动触发
- 克服人性弱点

## 五、思考与进化：网格策略的进阶

### 5.1 常见认知误区

**误区一："网格能100%赚钱"**

网格交易不是万能的，在单边行情中会失效甚至亏损。

**避坑**：选择震荡走势的标的，或在趋势行情中暂停网格。

**误区二："网格间距越小越好"**

间距太小会导致：
- 交易太频繁
- 手续费占比高
- 容易被震荡"扫死"

**避坑**：根据标的特点设置合理间距，一般5%左右。

**误区三："满仓网格最赚钱"**

满仓网格在极端下跌时可能亏光所有资金。

**避坑**：永远预留50%资金，只用一半仓位做网格。

### 5.2 网格策略进化

**进化一：不对称网格**

```
下跌间距：5%（跌得多买得多）
上涨间距：8%（涨得少卖得少）

优点：下跌时积累更多便宜筹码
```

**进化二：移动网格**

```
随着价格上涨，逐步提高基准价

优点：锁定更多利润
```

**进化三：趋势网格**

```
牛市：逐步提高网格上限，减少卖出
熊市：逐步降低网格下限，增加买入

优点：适应不同行情
```

### 5.3 与其他策略结合

**网格+定投**：

```
核心仓位（70%）：定投宽基指数
卫星仓位（30%）：网格交易ETF

收益来源：
├── 定投：长期复利收益
└── 网格：震荡行情额外收益
```

**网格+趋势**：

```
震荡期：用网格交易
趋势期：暂停网格，改用趋势策略

判断方法：
├── 连续3次触网 → 震荡市
└── 单边突破 → 趋势市
```

## 六、常见问题与避坑指南

### 6.1 Q&A

**Q1：网格交易手续费怎么算？**

A：买入万分之一，卖出千分之一。单次交易建议金额大于5000元，否则手续费占比过高。

**Q2：价格跌到网格下限怎么办？**

A：停止买入，等待反弹。可以设置"跌到下限后，涨回一格再买"的规则。

**Q3：价格涨到网格上限怎么办？**

A：停止卖出，等待下跌。可以设置"突破上限后，改用趋势策略"的规则。

**Q4：网格和定投有什么区别？**

A：定投是越跌越买，成本摊薄；网格是跌一格买一格，涨一格卖一格。定投适合所有行情，网格只适合震荡行情。

**Q5：可以用网格做股票吗？**

A：可以，但建议选择流动性好、不会退市的ETF或蓝筹股，避免踩雷。

### 6.2 避坑清单

| 坑 | 描述 | 避坑方法 |
|---|------|---------|
| 单边满仓 | 单边下跌时满仓被套 | 控制50%仓位 |
| 频繁交易 | 间距太小导致频繁买卖 | 设置5%左右间距 |
| 不会止损 | 跌破下限死扛 | 设定止损线 |
| 选错标的 | 选了单边下跌的标的 | 选波动适中的ETF |
| 不调整策略 | 趋势来了还死守网格 | 根据行情灵活调整 |

### 6.3 适合人群

| 适合 | 不适合 |
|------|--------|
| 有时间盯盘的投资者 | 没时间管理的 |
| 震荡市环境 | 单边趋势明显的市场 |
| 资金量适中（5-50万） | 资金量太大或太小 |
| 有一定投资经验的 | 完全零基础的 |
