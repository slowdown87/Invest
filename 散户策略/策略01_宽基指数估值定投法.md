## 一、零基础入门：什么是宽基指数定投

### 1.1 用买菜思维理解定投

想象你去菜市场买菜，土豆今天1元一斤，你买了100斤，花了100元。明天土豆降价到0.5元一斤，你又买了100斤，只花了50元。后天涨到2元一斤，你再买100斤，花了200元。三天下来，你买了300斤土豆，总共花了350元，平均成本是1.17元/斤。

这就是**定投的核心原理**：不管价格高低，定期买入固定金额，**价格低时自动买得多，价格高时自动买得少**，长期下来成本被平滑化，低于市场平均价格。

定投指数基金，就是**定投一篮子好股票**。你不需要研究个股，不需要担心某家公司暴雷倒闭，只需要相信中国经济会长期向上。

### 1.2 什么是宽基指数

**宽基指数**是指覆盖多个行业、选取多个代表性股票的指数，特点是不把鸡蛋放在一个篮子里。

| 指数名称 | 代表股票 | 特点 | 适合人群 |
|---------|---------|------|---------|
| **沪深300** | 茅台、宁德、招行等 | A股最核心的300只大盘蓝筹股 | 追求稳健的投资者 |
| **中证500** | 中小盘龙头股 | A股中盘股代表，成长性较强 | 平衡型投资者 |
| **创业板指** | 宁德、东方财富等 | 科技创新企业为主，波动大 | 进取型投资者 |
| **科创50** | 半导体、生物医药等 | 科创板核心科技股 | 高风险偏好者 |
| **中证A500** | 各行业龙头+细分龙头 | 覆盖更全面，行业均衡 | 长期配置者 |

### 1.3 什么是估值

**估值**就是"贵不贵"的判断。好东西便宜买才是好买卖。

想象你去买房子，同一个小区的房子，正常情况下值100万。但有时候市场恐慌，大家都急着卖，房价可能跌到70万；有时候市场狂热，房价可能被炒到130万。

**估值指标**（以市盈率PE为例）：

```
市盈率(PE) = 股价 ÷ 每股收益
```

- PE低 = 股价相对便宜 = 值得买入
- PE高 = 股价相对昂贵 = 考虑卖出

**估值百分位**是指当前估值在过去10年中的位置：
- 10%百分位：比90%的时候都便宜，极度低估
- 50%百分位：中间价格，合理区间
- 90%百分位：比90%的时候都贵，极度高估

### 2.1 第一性原理：两个收益来源

定投指数赚钱，来源于两个确定的收益来源：

**收益来源一：中国经济长期增长**

中国GDP从2000年的10万亿增长到2023年的126万亿，增长了约12倍。上市公司作为中国经济的主体，长期来看盈利持续增长，股价自然上涨。买入沪深300指数，就是买入中国经济的长期成长。

历史数据验证：沪深300指数从2005年成立至今（2024年），虽然经历多次牛熊转换，但底部不断抬高，长期趋势向上。

**收益来源二：市场估值周期性波动**

估值像钟摆一样，在悲观和乐观之间来回摆动：
- 熊市底部：PE可能低至10倍以下
- 牛市顶部：PE可能高达30-50倍

估值从低到高的回归过程，就是**额外的收益来源**。低估买入、高估卖出，可获取估值修复的收益。

### 2.2 均值回归理论

"树不会长到天上"——霍华德·马克斯

这句话同样适用于股市。价格短期内可能偏离价值，但长期必然回归价值。定投策略正是利用这一规律：

```
低估时买入 → 估值回归合理 → 高估时分批卖出
```

均值回归的关键前提：
1. 所投资产的价值必须长期存在（指数符合）
2. 低估和高估是周期性出现的（A股符合）
3. 投资者有足够的耐心等待回归（定投能做到）

### 2.3 定投的微笑曲线

很多人害怕市场下跌，认为下跌就是亏损。但对定投来说，下跌反而是"好事"——让你能以更低的价格买入更多份额。

**微笑曲线原理**：

```
时间 →

高点开始定投
↓ 不断买入，成本累积
↓ ↓
谷底  ←--- 此时买入最多，成本摊薄最多
↓ ↓
回升 → 成本低于平均价，开始盈利
```

无论从哪个位置开始定投，只要坚持到估值回升，都能盈利。这就是定投的"微笑曲线"魅力。

### 3.1 第一步：选择合适的指数基金

- 在证券账户中购买，像买卖股票一样

### 3.2 第二步：确定定投金额和频率

**定投金额**：建议为月收入的10%-20%，或每月固定金额（如每月1000-5000元）。

关键原则：**只用闲钱定投**，3-5年不用的钱，避免因急需用钱在低点割肉。

**定投频率**：
- 月定投：每月固定日期扣款，适合工薪族
- 周定投：每周固定日期扣款，成本更平滑
- 双周定投：折中方案

**实证数据**：月定投和周定投长期收益差异极小（约0.1%-0.2%），不必纠结频率。

### 3.3 第三步：判断估值——核心操作

**估值查询工具**：

1. **理杏仁**（免费）：https://www.lixinger.com
2. **且慢**（免费）：https://qieman.com
3. **天天基金网**（免费）：搜索指数温度
4. **支付宝-基金-指数估值**（免费）

**估值操作规则**：

```
┌─────────────────────────────────────────────────────────┐
│  估值百分位          │  操作策略                        │
├─────────────────────────────────────────────────────────┤
│  <10% (极度低估)      │  开启/加倍定投，捡便宜筹码        │
│  10%-30% (低估值)     │  正常定投，持续积累份额            │
│  30%-70% (合理区间)  │  停止定投，持有不动                │
│  70%-90% (高估值)     │  分批止盈，锁定部分利润            │
│  >90% (极度高估)      │  全部清仓，等待下一轮低估周期      │
└─────────────────────────────────────────────────────────┘
```

### 3.4 第四步：执行止盈——落袋为安

定投不卖，永远只是"纸上富贵"。止盈是定投最关键的一环。

**三种止盈方法**：

**方法一：估值止盈法（推荐）**
- PE百分位达到70%：止盈30%
- PE百分位达到80%：再止盈30%
- PE百分位达到90%：清仓

**方法二：目标收益率法**
- 设定目标收益率（如50%）
- 达到后分3批卖出（各卖1/3）
- 每上涨10%再卖1批

**方法三：最大回撤法**
- 从最高点下跌15%时全部卖出
- 优点：可能卖在更高位置
- 缺点：可能错过更大行情

### 3.5 第五步：记录与复盘

定投是长期事业，建议记录：
- 开始定投日期和指数点位
- 每期定投金额和净值
- 累计投入本金
- 当前市值和收益率

可以使用Excel或手机备忘录简单记录，每季度复盘一次。

## 五、思考与进化：定投策略的进阶之路

### 5.1 常见认知误区

**误区一："等跌到底再买"**

很多人想等"最低点"再开始定投，结果一等就是几年，错过红利期。

**真相**：没有人能准确预测最低点。定投的价值在于"不需要预测"，持续买入才是王道。

**误区二："亏了就停止定投"**

看到账户亏损，就停止扣款，想等市场好转再买。

**真相**：亏的时候停止定投，恰恰错过了积累便宜份额的机会。定投赚钱的核心就是"低位多买"。

**误区三："一次性全部投入"**

看到市场低估，把所有钱一次性买入。

**真相**：择时很难，一次性买入可能遭遇继续下跌。建议分6-12个月分批建仓。

### 5.2 定投策略的进化

**进化一：估值+低点多倍定投**

基础版：每月固定金额
进级版：估值<20%时，双倍定投；估值<10%时，三倍定投

优点：低位积累更多低成本份额
缺点：需要预留更多资金

**进化二：定投+网格增强**

在定投基础上，用小仓位（10%-20%）做网格交易，震荡行情额外增收

**进化三：核心+卫星配置**

核心仓位（70%）：宽基指数定投
卫星仓位（30%）：行业指数或主动基金，追求超额收益

### 5.3 如何面对至暗时刻

定投最难的不是技术，是心性。以下方法帮助你度过艰难时刻：

1. **心理准备**：开始定投前，务必做好"可能亏损20%-30%，持续1-2年"的心理准备

2. **不去看账户**：设置每月自动定投，然后关闭APP，不每天盯盘

3. **转移注意力**：定投是长期行为，把精力放在工作和生活上

4. **想象未来**：想象2-3年后市场好转时的收益，给自己打气

5. **寻找同伴**：加入定投社群，互相鼓励

## 六、常见问题与避坑指南

### 6.1 Q&A

**Q1：定投要不要止损？**
A：定投不止损，只止盈。止损是违背定投原理的。只要你买的是宽基指数，中国经济不会消失，坚持下去一定会盈利。

**Q2：定投多久能看到效果？**
A：A股一轮牛熊周期约3-5年，建议至少坚持3年以上。短期1-2年可能仍处于亏损状态。

**Q3：定投和一次性买入哪个更好？**
A：取决于资金性质和风险承受能力。一次性买入要求资金稳定、不惧波动；定投适合每月有固定收入的工薪族。

**Q4：同时定投多个指数可以吗？**
A：可以。建议不超过3个指数，资金分散到沪深300+中证500+创业板即可。太多指数反而分散精力。

**Q5：定投要择时吗？**
A：不需要精确择时。只要在估值低估或合理区间开始，坚持足够长时间，都能获得不错收益。

### 6.2 避坑清单

| 坑 | 描述 | 避坑方法 |
|---|------|---------|
| 高点all in | 在市场最热时把全部资金投入 | 分批建仓，不冲动 |
| 停止定投 | 亏损时停止扣款 | 设置自动扣款，不手动操作 |
| 不知止盈 | 只买不卖，坐过山车 | 严格执行估值止盈规则 |
| 急用钱 | 生活中急需用钱，被迫在低点赎回 | 只用闲钱，不影响生活 |
| 频繁切换 | 看到其他策略赚钱就换 | 坚持一种策略至少3年 |
