# 策略八：价值投资基本面选股法

> **长期复利核心策略 · 穿越牛熊的秘密武器 · 适合有研究能力的投资者**

## 一、零基础入门：什么是价值投资

### 1.1 用买房思维理解价值投资

想象你去买房：

```
好的买法（价值投资）：
├── 评估房子值多少钱（内在价值）
├── 市场上有房主急售，价格低于价值
├── 买入后等待价值被发现
└── 房子价格回归价值后卖出

不好的买法（投机）：
├── 看到别人在买，我也买
├── 房子涨了追进去
├── 跌了恐慌卖出
└── 最终亏损离场
```

**价值投资**就是：**用低于内在价值的价格买入好资产，等待价值被发现。**

### 1.2 核心概念：内在价值

**内在价值 = 未来能赚到的钱的总和**

想象你买了一只下金蛋的鹅：
- 这只鹅每年能下100个金蛋
- 每个金蛋值1元
- 这只鹅的内在价值就是每年100元

如果这只鹅现在卖50元：
- 价格 < 价值 → 值得买入

如果这只鹅现在卖200元：
- 价格 > 价值 → 不值得买入

### 1.3 价值投资的两个收益来源

```
收益来源一：企业成长
├── 好的公司，盈利会不断增长
├── 盈利增长 → 内在价值增长
└── 长期持有，伴随企业成长赚钱

收益来源二：估值修复
├── 市场悲观时，价格低于价值
├── 市场乐观时，价格高于价值
└── 低买高卖，赚估值修复的钱
```

## 二、专业原理：价值投资的三大基石

### 2.1 安全边际

**格雷厄姆的核心理念**

> 用4毛钱买价值1元的东西，即使判断错误，也不会亏太多。

**安全边际公式**：

```
安全边际 = (内在价值 - 买入价格) ÷ 内在价值

案例：
├── 内在价值：100元
├── 买入价格：60元
└── 安全边际：40%

相当于用6折价格买了100元的东西
```

### 2.2 能力圈原则

**只投资你看得懂的公司**

巴菲特说：
> 我们不必对所有行业都了解，只需要在自己能力圈范围内投资。

**什么是能力圈？**

```
看得懂的公司：
├── 你知道它是做什么的
├── 你能理解它的商业模式
├── 你能判断它能不能持续赚钱
└── 例如：你每天用的手机、银行、牛奶

看不懂的公司：
├── 复杂到看不懂的科技公司
├── 商业模式不清晰的区块链
└── 例如：某些高科技、军工企业
```

### 2.3 市场先生

格雷厄姆的"市场先生"比喻：

> 想象市场是一个情绪不稳定的人，每天跑来给你报价。有时他乐观过度，报价很高；有时他恐惧过度，报价很低。作为主人，你应该在他报价低时买进，在他报价高时卖出。

**市场先生的启示**：
- 市场短期波动不代表公司价值变化
- 市场恐慌时，反而是买入机会
- 市场狂热时，反而是卖出机会

## 三、实操流程与方法：选股三步法

### 3.1 第一步：选好行业

**什么是好行业？**

| 特征 | 解释 | 举例 |
|------|------|------|
| 弱周期 | 业绩不受经济波动影响 | 食品、医药 |
| 高ROE | 投入少、产出高 | 白酒、酱油 |
| 行业稳定 | 技术变化慢 | 银行、保险 |
| 龙头集中 | 赢家通吃 | 空调、手机 |

**不适合价值投资的行业**：

```
强周期行业：
├── 钢铁、煤炭：业绩随经济大起大落
├── 航运、航空：成本高、利润薄
└── 光伏、面板：技术迭代快，容易被颠覆

技术迭代快的行业：
├── 手机：技术更新快，龙头可能随时被颠覆
├── 互联网：赢家通吃，一旦落伍就完蛋
└── 新兴科技：风险太大
```

**推荐的行业**：

```
消费行业：
├── 白酒：贵州茅台、五粮液
├── 酱油：海天味业
├── 牛奶：伊利股份
└── 空调：格力电器、美的集团

金融行业：
├── 银行：招商银行、宁波银行
├── 保险：中国平安
└── 券商：中信证券

公用事业：
├── 电力：长江电力
├── 水务：重庆水务
└── 高速公路：宁沪高速
```

### 3.2 第二步：选好公司

**财务指标筛选**：

| 指标 | 标准 | 含义 |
|------|------|------|
| ROE | >15% | 股东回报率高 |
| 毛利率 | >30% | 定价能力强 |
| 净利润增速 | >10% | 成长性好 |
| 资产负债率 | <60% | 财务健康 |
| 分红率 | >30% | 愿意回馈股东 |

**护城河识别**：

```
无形资产护城河：
├── 品牌（茅台酒）
├── 专利（医药企业）
└── 特许经营（银行牌照）

转换成本护城河：
├── 换品牌代价高（微信、钉钉）
└── 习惯了不好换（office软件）

成本优势护城河：
├── 规模经济（大企业成本更低）
└── 地理优势（当地水务公司）
```

### 3.3 第三步：找好价格

**估值方法一：PE（市盈率）**

```
PE = 股价 ÷ 每股收益

用法：
├── PE低 = 相对便宜
├── PE高 = 相对贵
└── 同行业对比，PE越低越便宜

案例：
├── 茅台PE：30倍
├── 五粮液PE：25倍
└── 结论：五粮液相对茅台便宜（但还要看其他因素）
```

**估值方法二：PEG（市盈率相对盈利增长比率）**

```
PEG = PE ÷ 净利润增速

用法：
├── PEG < 1 = 成长性被低估
├── PEG = 1 = 合理估值
└── PEG > 1 = 成长性被高估

案例：
├── 公司PE：30倍
├── 净利润增速：40%
├── PEG = 30 ÷ 40 = 0.75
└── 结论：PEG<1，成长性被低估，值得关注
```

**估值方法三：PB（市净率）**

```
PB = 股价 ÷ 每股净资产

用法：
├── PB低 = 相对便宜
├── 适合金融、周期股
└── 不适合轻资产公司

案例：
├── 银行股PB：0.7倍（破净）
└── 结论：银行股很便宜，安全边际高
```

**估值方法四：股息率**

```
股息率 = 每股分红 ÷ 股价 × 100%

用法：
├── 股息率高 = 现金回报好
├── 股息率 > 5% = 很有吸引力
└── 适合稳健型投资者

案例：
├── 长江电力股息率：5%
├── 银行存款利率：2%
└── 结论：长江电力比存款更划算
```

## 四、详细案例：如何分析一家公司

### 4.1 案例标的：招商银行

**第一步：分析行业**

```
银行业特点：
├── 弱周期：经济好坏都要用银行
├── 刚需：存贷款是刚需
├── 牌照壁垒：不是谁都能开银行
└── 集中度高：强者恒强

行业评分：★★★★☆
```

**第二步：分析公司**

```
招商银行竞争优势：
├── 零售之王：个人业务最强
├── 资产质量好：坏账率低
├── 科技投入大：金融科技领先
└── 品牌口碑好：用户信任

财务数据（2023年）：
├── 营业收入：3391亿元
├── 净利润：1466亿元
├── ROE：17.06%
├── 净息差：2.23%
└── 不良贷款率：0.95%

财务评分：★★★★★
```

**第三步：分析价格**

```
估值数据（2024年）：
├── PE：8.5倍（历史平均约10倍）
├── PB：1.3倍（历史平均约1.5倍）
├── 股息率：4.8%
└── 估值水平：处于历史偏低位置

价格评分：★★★★☆
```

**第四步：综合结论**

```
招商银行综合评分：
├── 行业：★★★★☆
├── 公司：★★★★★
├── 价格：★★★★☆
└── 综合：★★★★☆（强烈推荐）

投资逻辑：
1. 银行业永续经营，刚需属性
2. 招行是银行中的茅台，竞争优势明显
3. 估值处于历史低位，安全边际充足
4. 4.8%股息率，比存款划算
```

### 4.2 操作计划

**买入计划**：

```
买入信号：
├── PE < 10倍（低估）
├── PB < 1.5倍（低估）
└── 出现恐慌性下跌

分批建仓：
├── 第一批（40%）：PE<10倍时买入
├── 第二批（30%）：再跌10%时买入
└── 第三批（30%）：再跌10%时买入

总仓位：
└── 单只股票不超过总资金20%
```

**持有计划**：

```
持有逻辑不变条件：
├── ROE持续>15%
├── 业绩稳定增长
├── 行业地位稳固
└── 估值不出现极端高估

卖出逻辑：
├── PE>15倍（高估）
├── 出现更好的标的
├── 发现判断错误
└── 基本面恶化
```

## 五、思考与进化：价值投资的深化

### 5.1 常见认知误区

**误区一："买了就长期持有不动"**

价值投资不等于买了就永远不卖。

**正确理解**：
- 基本面恶化 → 卖出
- 估值极端高估 → 卖出
- 发现更好的标的 → 换股

**误区二："低PE就是便宜"**

低PE可能是陷阱，周期股PE低的时候反而最危险。

**正确理解**：结合行业特性和增速看PE。

**误区三："只看财务数据"**

财务数据可能造假，需要多维度验证。

**正确理解**：财务数据 + 商业模式 + 行业地位综合判断。

### 5.2 价值投资的进化

**进化一：困境反转**

```
困境反转策略：
├── 好公司遇到暂时困难
├── 股价大幅下跌
├── 等困境解除，股价修复
└── 获取超额收益

案例：
├── 2013年茅台受限三公消费
├── 股价从200跌到100
├── 困境解除后涨到2000
└── 收益20倍
```

**进化二：成长价值**

```
成长价值策略：
├── 传统价值：买低估
├── 成长价值：买合理估值的好公司
├── 逻辑：好公司成长会消化高估值
└── 收益来源：企业成长

案例：
├── 贵州茅台：估值一直没便宜过
├── 但业绩增长30%/年
├── 持有10年收益30倍
└── 估值贡献少，成长贡献大
```

**进化三：组合价值**

```
构建价值投资组合：
├── 组合持有5-8只股票
├── 分散个股风险
├── 不同行业配置
└── 动态平衡

组合原则：
├── 单只不超过20%
├── 行业不超过30%
└── 每年检视一次
```

## 六、常见问题与避坑指南

### 6.1 Q&A

**Q1：价值投资需要多少资金？**

A：多少资金都可以做，但资金太少意义不大。建议至少5万元以上。

**Q2：需要多专业的财务知识？**

A：不需要成为财务专家。掌握基本的PE、ROE、毛利率分析即可。

**Q3：价值投资要持有多久？**

A：至少3-5年。价值发现需要时间。

**Q4：怎么看管理层？**

A：关注年报中的经营分析、管理层讲话、激励机制等。

**Q5：发现判断错误怎么办？**

A：立即止损，不要有侥幸心理。

### 6.2 避坑清单

| 坑 | 描述 | 避坑方法 |
|---|------|---------|
| 低PE陷阱 | 周期股低PE反而贵 | 看PB和增速 |
| 财务造假 | 数据可能不真实 | 多维度验证 |
| 线性外推 | 过去好不代表未来好 | 分析行业变化 |
| 过度集中 | 仓位太重 | 单只不超20% |
| 不止损 | 亏损死扛 | 设止损线 |

```
□ 理解了内在价值的概念

□ 有耐心持有3-5年的心理准备

□ 知道如何识别好公司
```
