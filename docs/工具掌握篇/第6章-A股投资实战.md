## 三、股票交易规则：必须掌握的基础

### 3.1 交易时间

**A股交易时段**：

| 时段 | 时间 | 说明 |
|------|------|------|
| 早盘集合竞价 | 9:15-9:25 | 可挂单可撤单 |
| 开盘定价 | 9:25-9:30 | 撮合成交 |
| 连续竞价 | 9:30-11:30 | 正常交易 |
| 午间休市 | 11:30-13:00 | 无法交易 |
| 连续竞价 | 13:00-15:00 | 下午交易 |
| 收盘集合竞价 | 14:57-15:00 | 最后撮合 |

**重要时间节点**：

| 时间 | 事件 |
|------|------|
| 9:15-9:20 | 挂单可撤，新股炒作常用 |
| 9:20-9:25 | 挂单不可撤，真实竞价 |
| 9:25 | 产生开盘价 |
| 9:30 | 正式开始交易 |
| 15:00 | 当日交易结束 |

### 3.2 交易单位

| 项目 | 规则 |
|------|------|
| 最小买卖单位 | 1手 = 100股 |
| 最小价格变动 | 0.01元（1分钱） |
| 买入限制 | 100股的整数倍 |
| 卖出限制 | 可以零股（不足100股） |

**交易数量示例**：

```
正确：
├── 买入100股 ✓
├── 买入200股 ✓
├── 买入1000股 ✓
└── 买入1500股 ✓

错误：
├── 买入150股 ✗（非100整数倍）
└── 买入99股 ✗（低于1手）
```

### 3.3 涨跌停制度

**各板块涨跌停幅度**：

| 板块 | 涨跌停 | ST股票 | 说明 |
|------|--------|--------|------|
| 主板（沪深） | ±10% | ±5% | 最常见 |
| 创业板 | ±20% | ±20% | 2020年改革 |
| 科创板 | ±20% | ±20% | 2019年设立 |
| 北交所 | ±30% | ±30% | 波动最大 |

**涨跌停计算示例**：

```
茅台（前收盘价100元）：
├── 涨停价：100 × 1.1 = 110元
└── 跌停价：100 × 0.9 = 90元

宁德时代（创业板，前收盘价200元）：
├── 涨停价：200 × 1.2 = 240元
└── 跌停价：200 × 0.8 = 160元
```

### 3.4 交易费用详解

**A股交易成本构成**：

```
总费用 = 佣金 + 印花税 + 过户费

各费用明细：
```

| 费用类型 | 收取方向 | 费率 | 最低 | 计算公式 |
|----------|----------|------|------|----------|
| 佣金 | 双向 | 约0.03% | 5元 | 成交金额 × 0.03% |
| 印花税 | 卖出 | 0.1% | 无 | 成交金额 × 0.1% |
| 过户费 | 双向 | 0.01% | 1元 | 成交金额 × 0.01%（仅沪市） |

**费用计算示例**：

```
场景：买入10万元沪市股票，卖出11万元

买入时：
├── 佣金：10万 × 0.03% = 30元（最低5元）
├── 印花税：0（买入不收）
├── 过户费：10万 × 0.01% = 1元
└── 买入总费用：31元

卖出时：
├── 佣金：11万 × 0.03% = 33元
├── 印花税：11万 × 0.1% = 110元
├── 过户费：11万 × 0.01% = 1.1元
└── 卖出总费用：144元

单次买卖总成本：175元（约0.16%）
```

### 📊 A股实战：佣金的影响

**案例：佣金差异的影响**

>
> ```
> 小明：佣金万3

> ```

## 四、买卖下单：第一次交易

### 4.1 委托方式

**市价委托 vs 限价委托**：

| 类型 | 说明 | 优点 | 缺点 |
|------|------|------|------|
| **市价委托** | 以当前最优价格成交 | 成交快 | 可能滑点 |
| **限价委托** | 指定价格买入/卖出 | 价格确定 | 可能不成交 |
| **挂单/撤单** | 挂出委托等待成交 | 可控制价格 | 需要等待 |

**限价委托示例**：

```
买入限价单：
├── 当前价格：100元
├── 挂单价格：95元
├── 委托数量：1000股
└── 如果跌到95元 → 自动成交

卖出限价单：
├── 当前价格：100元
├── 挂单价格：105元
├── 委托数量：500股
└── 如果涨到105元 → 自动成交
```

### 4.3 下单步骤详解

**首次买入操作流程**：

```
Step 1：打开券商APP，登录账户
    ↓
Step 2：搜索股票代码或名称
    ↓
Step 3：查看行情，确认买入价格
    ↓
Step 4：选择"买入"标签
    ↓
Step 5：输入买入数量（100股整数倍）
    ↓
Step 6：确认金额和费用
    ↓
Step 7：输入交易密码
    ↓
✅ 委托成功，等待成交
```

**卖出操作（类似）**：

```
不同点：
├── 选择"卖出"标签
├── 检查持仓数量
└── 注意印花税（卖出才收）
```

### 4.4 委托状态查询

**常见委托状态**：

| 状态 | 含义 |
|------|------|
| **待成交** | 委托已提交，等待撮合 |
| **部分成交** | 成交一部分，剩余等待 |
| **已成交** | 全部成交完成 |
| **已撤单** | 手动撤销或未成交撤销 |
| **已拒绝** | 委托不合规被拒绝 |

**如何撤单**：

```
当日有效：
├── 9:30-15:00内可撤单
├── 未成交的委托收盘后自动失效
└── 当日可以反复撤单重新挂

操作：
├── 在"当日委托"中查看
├── 点击要撤单的委托
└── 确认撤单
```

## 五、选股方法：找到值得买的股票

### 5.1 基本面选股：价值投资法

**核心指标筛选**：

| 指标 | 筛选标准 | 说明 |
|------|----------|------|
| ROE | >15% | 股东回报率高 |
| PE | <30 | 估值不能太贵 |
| 净利润增速 | >10% | 业绩持续增长 |
| 资产负债率 | <60% | 负债不能太高 |
| 毛利率 | >30% | 有竞争优势 |

**财务筛选示例**：

```
条件设置：
├── ROE > 20%
├── 净利润增速 > 15%
├── 资产负债率 < 50%
└── 上市时间 > 3年

筛选结果：
├── 贵州茅台：ROE 30%+ ✓
├── 海天味业：ROE 30%+ ✓
├── 宁德时代：ROE 20%+ ✓

### 5.2 技术面选股：趋势投资法

**趋势选股要点**：

| 要点 | 说明 |
|------|------|
| **均线多头排列** | 短期均线在长期均线上方 |
| **放量突破** | 成交量放大，突破重要价位 |
| **回踩均线** | 回调到均线附近获得支撑 |
| **MACD金叉** | DIF线上穿DEA线 |

**技术面选股示例**：

```
信号识别：
├── 5日均线上穿10日均线 → 金叉买入信号
├── 股价回踩20日均线 → 支撑买入点
├── 成交量放大2倍以上 → 资金关注
└── MACD在0轴上方 → 强势市场
```

## 六、股票池管理：建立自己的股票库

### 6.1 为什么要建立股票池

**股票池的意义**：

```
没有股票池的问题：
├── 盘中临时起意买股票
├── 听消息跟风买入
├── 买自己不了解的公司
└── 结果：亏损概率大

有股票池的好处：
├── 提前研究，胸有成竹
├── 等待好价格出现
├── 减少冲动交易
└── 提高投资胜率
```

### 6.2 股票池分类

**三层股票池**：

```
核心池（3-5只）：
├── 最看好的优质公司
├── 可以重仓持有
└── 长期跟踪

卫星池（5-10只）：
├── 阶段性机会
├── 行业配置
└── 中期持有

观察池（10-20只）：
├── 持续关注的标的
├── 等待好价格
└── 短线机会
```

**股票池维护**：

| 维护内容 | 频率 |
|----------|------|

## 七、仓位管理：控制风险的艺术

### 7.1 仓位的基本概念

**仓位计算**：

```
仓位 = 股票市值 ÷ 总资金 × 100%

示例：
├── 总资金：100万
├── 持有茅台股票：30万
└── 仓位 = 30万 ÷ 100万 = 30%
```

**仓位类型**：

| 仓位 | 占比 | 含义 |
|------|------|------|
| 空仓 | 0% | 全部是现金 |
| 轻仓 | 20-30% | 小试牛刀 |
| 半仓 | 50% | 进可攻退可守 |
| 重仓 | 70-80% | 大部分资金入市 |
| 满仓 | 100% | 全部资金入市 |

### 7.2 仓位管理原则

**新手仓位建议**：

| 经验 | 建议仓位 | 说明 |
|------|----------|------|
| 模拟盘阶段 | 0% | 先学习 |
| 实盘第一年 | 20-30% | 小资金学习 |
| 有一定经验 | 50% | 半仓操作 |
| 形成体系后 | 70-80% | 谨慎满仓 |

**分批建仓法**：

```
金字塔建仓：
├── 第一次买入：20%（底仓）
├── 第二次买入：15%（确认后加仓）
├── 第三次买入：10%（再加仓）
└── 保留现金：55%（应对下跌）

优点：
├── 成本逐步抬高
├── 下跌时有资金补仓
└── 降低一次性买入的风险
```

### 7.3 止损原则

**为什么要止损**：

```
不止损的后果：
├── 亏损10% → 需要涨11%回本
├── 亏损20% → 需要涨25%回本
├── 亏损50% → 需要涨100%回本
└── 亏损80% → 基本回本无望

止损的意义：
├── 截断亏损
├── 保住本金
└── 等待下次机会
```

**止损方法**：

| 方法 | 说明 | 适用 |
|------|------|------|
| 固定比例止损 | 亏损10%止损 | 短线 |
| 均线止损 | 跌破均线止损 | 趋势交易 |
| 逻辑止损 | 基本面恶化止损 | 长线 |

**止损示例**：

```
买入价格：100元
止损设置：亏损10%止损

下跌到90元 → 触发止损 → 卖出

如果不止损：
├── 跌到80元 → 亏损20%
├── 跌到50元 → 亏损50%
└── 可能永远解不了套
```

**题目1：A股交易时间和规则特点**

**参考答案**：

**交易时间特点**：

```
A股交易时间：9:30-11:30，13:00-15:00

对比其他市场：
├── 美股：北京时间22:30-次日凌晨5:00
├── 港股：北京时间9:30-12:00，13:00-16:00
└── A股：作息正常，便于盯盘

关键时间点：
├── 9:15-9:20：可挂可撤（新股炒作常用）
├── 9:20-9:25：只挂不撤（真实竞价开始）
├── 9:25：产生开盘价
└── 15:00：当日交易结束
```

**涨跌停制度的意义**：

```
涨跌停限制：±10%（主板）/ ±20%（科创板/创业板）

目的：
1. 防止过度炒作
   └── 避免单日涨幅过大

2. 保护投资者
   └── 限制单日最大亏损

3. 市场降温
   └── 给投资者冷静思考时间

4. 维护稳定
   └── 避免踩踏式暴跌
```

**实际影响**：

```
涨停股特点：
├── 次日往往高开（资金博弈）
├── 可能连续涨停（妖股）
└── 流动性降低（卖不出）

跌停股特点：
├── 次日往往低开
├── 流动性枯竭
└── 踩踏风险
```

**题目2：市价委托 vs 限价委托**

**参考答案**：

**市价委托**：

```
定义：以当前市场价格立即成交

特点：
├── 成交速度快
├── 价格不确定
└── 可能"滑点"

适用场景：
├── 急着买入/卖出
├── 波动剧烈的股票
└── 流动性好的股票

示例：
├── 涨停板买入 → 必须市价
└── 跌停板卖出 → 必须市价
```

**限价委托**：

```
定义：指定价格买入/卖出

特点：
├── 价格确定
├── 可能不成交
└── 不会买贵/卖便宜

适用场景：
├── 稳健交易
├── 有明确目标价
└── 流动性一般的股票

示例：
├── 100元买入 → 只能≤100元成交
└── 110元卖出 → 只能≥110元成交
```

**对比表**：

| 对比项 | 市价委托 | 限价委托 |
|--------|----------|----------|
| 成交速度 | 快 | 不确定 |
| 价格确定 | 不确定 | 确定 |
| 成交保证 | 有 | 无 |
| 滑点风险 | 有 | 无 |
| 适合场景 | 紧急交易 | 稳健交易 |

**实操建议**：

```
新手建议：
├── 优先使用限价委托
├── 设置合理的价格
├── 养成挂单习惯
└── 不要追涨杀跌

进阶后：
├── 根据情况选择
├── 市价委托用于止损
└── 限价委托用于建仓
```

**题目3：为什么要建立股票池？入池条件？**

**参考答案**：

**为什么要建立股票池**：

```
没有股票池的问题：
├── 盘中临时看消息买入
├── 听别人推荐买入
├── 追热点跟风买入
├── 买完全不了解的公司
└── 结果：亏损概率>80%！

有股票池的好处：
├── 提前做好功课，胸有成竹
├── 等待好价格出现才买
├── 减少冲动交易
├── 提高投资胜率
└── 便于持续跟踪
```

**股票入池标准**：

```
一、基本面标准
├── ROE > 15%（股东回报）
├── 净利润增速 > 10%（成长性）
├── 毛利率 > 30%（竞争优势）
├── 资产负债率 < 60%（财务健康）
└── 经营现金流为正（真赚钱）

二、行业标准
├── 行业龙头（前3名）
├── 有护城河（品牌/技术/规模）
├── 行业前景好
└── 不在夕阳行业

三、估值标准
├── PE < 30（不过于昂贵）
├── 处于历史估值低位更好
└── 有安全边际

四、流动性标准
├── 日成交额 > 1亿
├── 不是小盘仙股
└── 大股东质押比例不高
```

**股票池维护**：

```
定期检查：
├── 每日：持仓股行情
├── 每周：行业动态
├── 每月：股票池更新
└── 每季：财务数据重评

退出机制：
├── 基本面恶化
├── 估值过高
├── 行业变差
└── 发现更好的标的
```

**题目4：止损的重要性和方法**

**参考答案**：

**为什么要止损**：

```
数学道理：
├── 亏10% → 需赚11%回本
├── 亏20% → 需赚25%回本
├── 亏30% → 需赚43%回本
├── 亏50% → 需赚100%回本
└── 亏80% → 需赚400%回本！

越不止损，越难回本！

现实教训：
├── 乐视：从44元跌到0.18元
├── 中石油：从48元跌到4元
└── 持有不卖 → 永久亏损
```

**止损的意义**：

```
止损的作用：
├── 截断亏损
├── 保住本金
├── 保留实力
├── 等待下次机会
└── 提高长期胜率

不止损的代价：
├── 本金永久损失
├── 错失其他机会
├── 影响投资心态
└── 陷入长期套牢
```

**常用止损方法**：

| 方法 | 说明 | 适用 |
|------|------|------|
| **固定比例止损** | 亏损5-10%止损 | 短线 |
| **均线止损** | 跌破10日/20日均线 | 趋势交易 |
| **支撑位止损** | 跌破重要支撑位 | 技术分析 |
| **逻辑止损** | 基本面恶化止损 | 长线投资 |

**止损设置示例**：

```
短线止损（5-7%）：
├── 买入价：100元
├── 止损价：93元（-7%）
└── 严格执行

趋势止损（均线）：
├── 买入价：100元
├── 10日均线：95元止损
└── 跌破即卖

逻辑止损：
├── 买入逻辑：业绩增长
├── 止损逻辑：季度业绩下滑
└── 不看价格，看逻辑
```

**止损执行要点**：

```
✅ 止损要提前设好，不要临时决定
✅ 设定后要严格执行
✅ 止损不是失败，是保护
✅ 一次止损，换来长期生存

❌ 亏损时不止损，想等回本
❌ 把止损位一改再改
❌ 因为亏太多不舍得卖
❌ 想"高抛低吸"摊薄成本
```
